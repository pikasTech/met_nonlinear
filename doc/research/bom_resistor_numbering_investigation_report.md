# BOM电阻编号对应关系深度调查报告

## 执行摘要

本报告深入调查了权重电阻BOM（Bill of Materials）中R1-R266的编号顺序，详细分析了每个电阻编号对应的神经网络层、通道、权重类型等信息。通过对`all_layers_resistances_bom.csv`和`all_layers_resistances.csv`的对比分析，完整解析了BOM编号体系。

## 调查背景

在成功实现BOM导出功能后，需要深入理解电阻编号与神经网络结构的对应关系，以便：
1. 确保PCB设计和制造的准确性
2. 便于故障诊断和调试
3. 为自动化贴装编程提供参考
4. 验证差分电路实现的完整性

## BOM编号规则分析

### 1. 筛选规则

根据`weight_resistor_bom_generator.py`的实现逻辑：
```python
weight_df = df[df['type'].isin(['input_pos', 'input_neg', 'bias_pos', 'bias_neg'])].copy()
```

BOM只包含以下四种类型的电阻：
- **input_pos**: 输入权重正端电阻
- **input_neg**: 输入权重负端电阻  
- **bias_pos**: 偏置权重正端电阻
- **bias_neg**: 偏置权重负端电阻

### 2. 编号顺序

电阻按照在原始CSV文件中的出现顺序重新编号：
- 保持层(layer)的顺序：layer2 → layer3 → layer4 → layer5
- 每层内按通道(channel)顺序：channel 1 → 2 → 3 → ... → 6
- 每通道内按类型和索引顺序：input对(index 1-6) → bias对

## 关键发现

### 1. 差分对实现模式

每个神经网络权重参数对应一对电阻（正端/负端），形成差分实现：
- **正权重**：input_neg有效值，input_pos开路（1000MΩ）
- **负权重**：input_pos有效值，input_neg开路（1000MΩ）
- **零权重**：两端都开路（极少见）

### 2. 开路电阻的意义

1000MΩ（1GΩ）表示开路状态，在133个权重电阻对中：
- **133个开路电阻**：每对中总有一端开路（差分实现特性）
- **133个有效电阻**：范围从226Ω到621kΩ，包含实际权重信息

### 3. 偏置电阻特征

19个通道中的偏置电阻分布：
- 部分通道偏置正端开路，负端有效
- 部分通道偏置负端开路，正端有效
- 偏置值范围：21.9kΩ到621kΩ（有效值）

### 4. Layer4的特殊性

Layer4只有1个通道（14个电阻），而其他层都有6个通道。这反映了神经网络结构的特殊设计：
- Layer2: 6维输入 → 6通道输出
- Layer3: 6维输入 → 6通道输出
- Layer4: 6维输入 → 1通道输出（维度缩减）
- Layer5: 1维输入 → 6通道输出（维度扩展）

## 编号查询快速参考

给定BOM编号R(n)，可以快速定位：

```
层号 = floor((n-1) / 84) + 2  # 对于layer2-3
如果 n >= 169:
    if n <= 182: layer4
    else: layer5
    
通道号 = ((n-1) % 84) / 14 + 1  # 对于有6通道的层

通道内位置 = (n-1) % 14
    0-11: input权重对 (索引1-6)
    12-13: bias权重对
```

## 实际应用建议

### 1. PCB设计考虑
- 相邻奇偶编号形成差分对（如R1-R2, R3-R4）
- 布局时应保持差分对的对称性
- 开路电阻可以用DNP（Do Not Populate）标记

### 2. 故障诊断
- 快速定位：通过编号直接找到对应的网络层和通道
- 差分对验证：检查每对中是否恰有一个开路
- 阻值范围检查：有效值应在226Ω-621kΩ范围内

### 3. 自动化贴装
- 1000MΩ电阻可以不贴装（节省成本）
- 编程时可根据奇偶性判断差分对关系
- Layer4的特殊处理（只有14个电阻）

## 结论

BOM电阻编号系统采用严格的顺序编号，完全反映了神经网络的层次结构。266个电阻准确对应133个神经网络参数的差分实现，编号体系清晰、可追溯，为PCB制造和调试提供了良好的基础。建议在PCB丝印和制造文档中保留这种编号体系，以便于后续的调试和维护工作。
