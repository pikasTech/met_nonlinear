### R6: E96量化基础设施调查

**任务**: 调查SVF-DENSE单层仿真中如何实现电阻的E96量化误差

**关键发现**:
1. **E96系列定义**: 在 `spice_simulator/circuit_base.py` 中定义，共96个标准值
2. **量化方法**: `_convert_to_standard_value()` 方法实现浮点到E96的转换
3. **配置传递**: `inference_config.use_e96` 配置通过多层传递到电路层

**核心代码** (`circuit_base.py`):
```python
E96_VALUES: List[float] = [
    1.00, 1.02, 1.05, 1.07, 1.10, 1.13, 1.15, 1.18, 1.21, 1.24, 1.27, 1.30,
    # ... 共96个值
    9.53, 9.76
]

def _convert_to_standard_value(self, value: float) -> float:
    """将任意电阻值转换为最接近的E96标准值"""
    if value <= 0:
        return value
    exponent = np.floor(np.log10(value))
    mantissa = value / (10 ** exponent)
    closest_value = min(self.E96_VALUES, key=lambda x: abs(x - mantissa))
    return closest_value * (10 ** exponent)
```

```python
            abs_path = os.path.normpath(os.path.join(
                os.path.dirname(report_path), img_path
            ))

        if not os.path.exists(abs_path):
            issues.append(img_path)

    return issues
```
