### 阶段 2: 核心接口实现 (P1)

#### 2.1 TensorFlow 后端实现

**文件**: `src/backends/tensorflow.py`

**实现内容**:
| 类名 | 职责 | 依赖 |
|-----|------|-----|
| `TFDataLoader` | TF Dataset API 封装 | `tf.data` |
| `TFModel` | 适配现有模型到 BaseModel | `tf.keras.Model` |
| `TFOptimizer` | 适配 `tf.keras.optimizers` | `tf.optimizers` |
| `TFLoss` | 适配 `loss_functions.py` | 自定义损失函数 |
| `TFLRScheduler` | 适配 `LearningRateScheduler` | `tf.keras.callbacks` |
| `TFCallback` | 适配 `RealTimeTrainingCallback` | 自定义回调 |

**关键适配点**:
```python
# TFModel 适配现有模型示例
class TFModel(BaseModel):
    def __init__(self, model_type, model_config):
        self.model = self._create_model(model_type, model_config)

    def _create_model(self, model_type, config):
        if model_type == 'wavenet5':
            return WaveNet5(**config)
        elif model_type == 'lstm':
            return LSTM(**config)
        # ... 其他模型
```

#### 2.2 ModelEngine 重构

**文件**: `src/core/model_engine.py`

**修改方案**:
```python
class ModelEngine:
    def __init__(self, config, project_path=None):
        # ... 现有初始化 ...
        self.backend = create_backend(config.ml_backend)
        self.data_loader = self.backend.create_data_loader(config)
        self.model = self.backend.create_model(config)
        self.optimizer = self.backend.create_optimizer(config)
        self.loss = self.backend.create_loss(config)
```

**需要修改的方法**:
- `load_dataset()` - 使用 `self.backend.data_loader`
- `build_model()` - 使用 `self.backend.model`
- `train_model()` - 使用 `self.backend.optimizer` 和 `self.backend.loss`
- `evaluate_loss()` - 使用 `self.backend.loss`

### 阶段 3: 模型迁移 (P2)

#### 3.1 现有模型重构

**文件**: `src/models/base_models.py`, `frikan_models.py`, `wavenet_models.py`

**策略**: 模型定义保持不变，通过适配器模式接入新后端

```
方案 A: 保持现有模型类不变
├── 优点: 改动最小，向后兼容
└── 缺点: TF/PyTorch 模型类需分别维护

方案 B: 创建抽象模型定义
├── 优点: 模型定义统一
└── 缺点: 实现复杂，需要重写所有模型
```

**推荐方案 A**: 现有模型类不变，通过 `TFModel` 和 `PTModel` 适配器封装

#### 3.2 损失函数抽象

**文件**: `src/core/loss_functions.py`

**抽象方案**:
```python
# src/backends/tensorflow.py
class TFLoss(BaseLoss):
    def __init__(self, loss_name):
        self.loss_fn = self._get_tf_loss(loss_name)

    def _get_tf_loss(self, name):
        if name == 'power_log_mae':
            return power_log_mae_loss
        elif name == 'power_log':
            return power_log_loss
        # ...

    def __call__(self, y_true, y_pred):
        return self.loss_fn(y_true, y_pred)
```
