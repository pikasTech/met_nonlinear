## 文献调研重点方向

### P0 - 必须覆盖的基础理论

1. **Wiener 模型与非线性系统识别**
   - Wiener 系统的基本理论
   - Wiener 模型在电化学传感器/地震计中的应用
   - Wiener 与其他非线性模型（Hammerstein、Polynomials）的比较
   - **关键**：Wiener 的线性部分对应 RNN，非线性部分对应 KAN 的理论联系

2. **KAN (Kolmogorov-Arnold Networks)**
   - 原始 KAN 论文及理论证明
   - KAN 与 MLP 的本质区别
   - KAN 的计算效率特性（特别是 LUT 实现）
   - KAN 在时序数据/非线性建模中的应用

3. **AFMAE 损失函数相关理论**
   - 频域损失函数在时间序列中的应用
   - MAE vs MSE vs 频域损失的对比研究
   - AFMAE 的理论基础

### P1 - 必须覆盖的对比方法文献

4. **深度学习频率漂移补偿相关工作**
   - 电化学传感器/地震计信号漂移补偿
   - 功率放大器线性化中的深度学习方法
   - 与反馈补偿的比较研究

5. **神经网络架构的计算效率对比**
   - RNN vs 1D-CNN 的参数量和效率对比
   - KAN vs LSTM/GRU/Transformer 的计算效率对比
   - 不同激活函数的计算特性（ReLU、tanh、B-spline）

6. **数据集构建参考**
   - 电化学/地震信号数据集的构建方法
   - 数据集发布论文的标准化描述
   - 数据集划分和预处理的最佳实践

### P2 - 可选扩展方向

7. **计算效率评估方法**
   - 参数量、MACs、推理时间的标准评估方法
   - LUT 实现的相关工作

8. **公开数据集**
   - 电化学/地震信号公共数据集

## 调研原则

- **理论支撑优先**：每项声称都需要充分的文献支撑
- **对比全面性**：确保与领域内主要方法进行比较
- **可追溯性**：所有文献必须有完整的引用信息
- **系统性**：按理论类别组织文献，形成完整的知识图谱
- **聚焦效率**：重点关注计算效率对比，而非精度对比
