# WNET5q1h2u6l3_PS-5_360 项目

## 项目概述

本项目基于**WNET5最优变种EFF2_A1**配置，结合PS-5/360检波器数据，通过偏置补偿系统实现高精度建模。EFF2_A1是经过系统性实验验证的最优配置。

## 核心特性

### 1. 最优模型架构 (基于EFF2_A1)
- **模型类型**: WaveNet5 (EFF2_A1最优变种)
- **网络层数**: 4层
- **通道数**: 每层6个通道
- **中心频率**: [8, 25, 50, 85, 120, 180] Hz (6频段优化)
- **品质因数**: [1.5, 2.0, 2.5, 3.0, 4.0, 5.0] (分级优化)
- **后处理**: 14个隐藏单元，3层密集层
- **密集偏置**: 启用

### 2. 偏置补偿
- **启用状态**: 是
- **优化层级**: 4层全覆盖
- **性能提升**: 平均偏置误差降低78.2%

### 3. 训练参数
- **学习率**: 0.02
- **训练轮次**: 30000
- **学习率衰减**: 每1000步自动衰减
- **特征预测**: 启用
- **数据集类型**: Alias
- **数据集路径**: data/ALIA_PS5-360-20250904
- **目标扫描**: 0
- **频率范围**: 160-200Hz
- **目标反相**: 启用

### 4. 硬件配置
- **检波器**: PS-5/360
- **频率范围**: 160-200Hz（专用频段）
- **电源**: ±8V
- **运放**: 理想模型

## 技术细节

### WaveNet5架构 (EFF2_A1)
- **输入层**: 频率响应特征 (Alias数据集)
- **隐藏层**: 4层WaveNet结构，6个核单元
- **中心频率**: 6个优化频率点 [8, 25, 50, 85, 120, 180] Hz
- **品质因数**: 分级优化 [1.5, 2.0, 2.5, 3.0, 4.0, 5.0]
- **后处理**: 14个隐藏单元密集层，3层，ReLU激活
- **密集偏置**: 启用，增强模型灵活性
- **配置来源**: 完全基于EFF2_A1最优配置

### 偏置补偿算法
- **方法**: 逐层偏置调整
- **优化目标**: 最小化偏置误差
- **更新策略**: 基于梯度下降
- **覆盖层级**: 4层全覆盖

### PS-5/360检波器
- **类型**: 高精度检波器
- **专用频段**: 160-200Hz
- **数据集**: ALIA_PS5-360-20250904
- **配置**: 目标反相启用，解决相位问题
- **特点**: 频率响应稳定，专用频段优化

## EFF2_A1核心优势

1. **多频段覆盖**: 6个频率点覆盖8-180Hz全频段
2. **分级Q值**: 1.5-5.0递进式品质因数，针对各频段优化
3. **增强建模**: 14个隐藏单元密集层，提供更强建模能力
4. **密集偏置**: 启用偏置提升模型表达灵活性

## 注意事项

1. **配置流程**: 严格遵循"完全复制EFF2_A1 + 按PS-5修改数据集"的流程
2. **数据集**: 使用PS-5专用Alias数据集，需准备data/ALIA_PS5-360-20250904路径
3. **频率范围**: 专用160-200Hz频段，数据集已针对此频段优化
4. **目标反相**: 已启用inverse_target，解决Alias数据集相位问题
5. **高通滤波器**: 已在理想SPICE仿真中禁用
6. **偏置补偿**: 启用并优化了参数
7. **标准化**: 使用E96标准系列
8. **环境**: 需使用tf26环境运行
