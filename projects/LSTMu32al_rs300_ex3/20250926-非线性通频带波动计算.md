# 通频带波动计算方法

## 1. 符号定义与通用公式

设灵敏度样本集合为 \(\{S_i\}\)。定义：

1. 最大 / 最小灵敏度：
$$S_{\max} = \max_i S_i, \qquad S_{\min} = \min_i S_i.$$
2. 线性峰峰值（线性波动幅度）：
$$\Delta S = S_{\max} - S_{\min}.$$
3. dB 波动（幅度型灵敏度的对数波动）：
$$R_{\text{dB}} = 20\log_{10}\left(\frac{S_{\max}}{S_{\min}}\right).$$
4. 补偿前后量（用上标 \(\text{orig}\) 与 \(\text{comp}\) 区分）：
$$S^{\text{orig}}_{\max},\; S^{\text{orig}}_{\min},\; \Delta S_{\text{orig}},\; R^{\text{orig}}_{\text{dB}}$$
$$S^{\text{comp}}_{\max},\; S^{\text{comp}}_{\min},\; \Delta S_{\text{comp}},\; R^{\text{comp}}_{\text{dB}}.$$
5. 线性峰峰值抑制率：
$$\eta_{\text{linear}} = \frac{\Delta S_{\text{orig}} - \Delta S_{\text{comp}}}{\Delta S_{\text{orig}}} \times 100\%.$$
6. dB 波动抑制率：
$$\eta_{\text{dB}} = \frac{R^{\text{orig}}_{\text{dB}} - R^{\text{comp}}_{\text{dB}}}{R^{\text{orig}}_{\text{dB}}} \times 100\%.$$

## 2. 数值代入计算

根据补偿前后的灵敏度峰值计算，补偿前后的峰值如下：

$$S^{\text{orig}}_{\max} = 260.192135, \qquad S^{\text{orig}}_{\min} = 155.074964$$
$$S^{\text{comp}}_{\max} = 209.036241, \qquad S^{\text{comp}}_{\min} = 189.679038$$

线性峰峰值：
$$\Delta S_{\text{orig}} = 260.192135 - 155.074964 = 105.117172$$
$$\Delta S_{\text{comp}} = 209.036241 - 189.679038 = 19.357204$$

线性抑制率：
$$\eta_{\text{linear}} = \frac{105.117172 - 19.357204}{105.117172} \times 100\% = 81.59\%.$$

dB 波动：
$$R^{\text{orig}}_{\text{dB}} = 20\log_{10}\left(\frac{260.192135}{155.074964}\right) = 4.495050\,\text{dB}$$
$$R^{\text{comp}}_{\text{dB}} = 20\log_{10}\left(\frac{209.036241}{189.679038}\right) = 0.844045\,\text{dB}$$

dB 抑制率：
$$\eta_{\text{dB}} = \frac{4.495050 - 0.844045}{4.495050} \times 100\% = 81.22\%.$$
