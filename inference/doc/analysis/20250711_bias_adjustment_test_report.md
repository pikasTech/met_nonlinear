## 发现的问题和建议

### 问题1: 误差分析器兼容性
**描述**: `error_analyzer.py` 不支持部分层推理验证  
**影响**: 无法使用 `cli.py -a` 进行偏置误差分析  
**建议**: 更新错误分析器以支持 `--layers` 参数

**代码位置**: `inference/management/error_analyzer.py:233`
```python
# 当前代码：
if actual_layers != expected_layers:
    error_msg = f"{model_name}应该有{expected_layers}层，但数据显示{actual_layers}层。\n推理过程可能有错误。"
    raise ValueError(error_msg)

# 建议修改：
if layers_param is None and actual_layers != expected_layers:
    error_msg = f"{model_name}应该有{expected_layers}层，但数据显示{actual_layers}层。\n推理过程可能有错误。"
    raise ValueError(error_msg)
elif layers_param is not None and actual_layers != layers_param:
    error_msg = f"指定推理{layers_param}层，但数据显示{actual_layers}层。"
    raise ValueError(error_msg)
```
