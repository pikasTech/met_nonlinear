## 5. 问题与修复

### 问题1：SPICE和NumPy后端未遵守层数限制
- **发现时间**：初始测试阶段
- **问题描述**：指定`--layers 2`时，NN后端正确限制为2层，但SPICE和NumPy后端仍处理所有5层
- **根因分析**：
  1. `InferenceExecutor._execute_spice_inference()`方法未传递`layers`参数
  2. `SPICEBackend.infer_unified()`方法未从kwargs中提取`layers`参数
- **修复方案**：
  1. 在`InferenceExecutor`中添加`layers=self.layers_limit`参数传递
  2. 在`SPICEBackend.infer_unified()`中提取并传递`layers`参数
- **修复状态**：✅ 已修复并验证
