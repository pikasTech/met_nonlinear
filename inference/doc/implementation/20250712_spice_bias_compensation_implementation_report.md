## 三、关键技术点

### 3.1 正确的补偿应用流程

```
配置文件 → WaveNet5Model → WaveNet5SPICEBackend
                               ↓
                         export_model_to_spice()
                               ↓
                    _prepare_bias_compensations()
                               ↓
                   _apply_compensations_to_layers()
                               ↓
                      layer._temp_bias_compensation
                               ↓
                        DenseLayer.to_spice()
                               ↓
                    bias_vector + compensation
                               ↓
                        SPICE 电路（调整后）
```

### 3.2 补偿机制的核心原理

1. **NN 作为基准**：神经网络输出是标准答案（误差 = 0）
2. **SPICE 需要校准**：通过调整电路参数来匹配 NN 输出
3. **补偿仅影响 SPICE**：不修改 NN 推理的任何部分

## 四、遇到的问题和解决方案

**问题**：在推理前应用补偿
