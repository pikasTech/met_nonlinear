# Dense Circuit 偏置滤波完整方案

## 项目概述

本项目实现了在 `circuit_dense` 的加法器输出与激活函数输入之间添加高通滤波器，用于补偿运放偏置误差的完整解决方案。

### 核心问题
- 加法器电路已包含bias电压，但运放的非理想特性（偏置电压、偏置电流等）导致实际输出存在误差
- 需要在激活函数（ReLU）之前进行直流偏置补偿，而不修改激活函数电路本身

### 解决方案
在加法器输出 (`out{ch}_pre`) 与激活函数输入之间插入无源高通滤波器：
```
[加法器输出] ----C---- [激活函数输入]
                  |
                  R
                  |
             [bias电压]
```

## 技术要点

### 优势
✅ **非侵入性**：不修改现有激活函数电路  
✅ **自适应补偿**：无需精确知道偏置误差大小  
✅ **简单可靠**：仅需RC无源元件  
✅ **灵活配置**：支持通道级和全局配置  
✅ **向后兼容**：默认禁用，不影响现有代码  

### 关键参数
- **推荐截止频率**：0.1Hz - 1Hz
- **推荐配置**：R=1MΩ, C=1.6μF (fc=0.1Hz)
- **建立时间**：约5τ = 8秒达到99%

### 适用场景
- 高精度模拟计算需求
- 多级级联系统中的偏移累积控制
- 需要稳定直流工作点的应用
- 对偏置精度要求严格的神经网络硬件实现
