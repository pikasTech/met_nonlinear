# Bias Tuner - Automated SPICE Bias Compensation Tool

A zero-dependency tool for optimizing bias compensation parameters in neural network SPICE implementations.

## Features

- **Sequential Compensation**: Automatically applies compensation layer-by-layer with re-measurement
- **Multiple Strategies**: Supports same-phase, scaled, adaptive, and conservative compensation
- **Mock Mode**: Full testing support without requiring actual cli.py execution
- **Comprehensive Logging**: Detailed progress tracking and error reporting
- **JSON-based Communication**: Clean interface through config.json and error_analysis.json

## Installation

No installation required! The tool is designed to be zero-dependency and works with standard Python 3.

```bash
# Clone or copy the bias_tuner directory to your project
cp -r bias_tuner /path/to/your/project/
```

## Quick Start

```python
from bias_tuner import BiasTuner, CompensationStrategy

# Create tuner
tuner = BiasTuner(
    project_path="/path/to/WNET5q1h2u6l3",
    strategy=CompensationStrategy.SAME_PHASE,
    python_env="conda run -n tf26 python"
)

# Run baseline measurement
baseline = tuner.run_baseline_measurement()

# Tune layers sequentially
results = tuner.tune_sequential(layer_order=[1, 2, 3])

# Generate report
report_path = tuner.generate_report()
```

## Architecture

### Core Modules

- **ConfigManager**: Handles project configuration and bias compensation settings
- **BiasAnalyzer**: Extracts and analyzes bias errors from error_analysis.json
- **BiasCompensator**: Calculates optimal compensation values using various strategies
- **CommandExecutor**: Manages subprocess calls to cli.py with mock support
- **BiasTuner**: Main orchestrator that coordinates the tuning workflow

### Compensation Strategies

1. **SAME_PHASE**: Compensates with same sign as error (experimentally validated)
2. **SCALED**: Applies a scaling factor to compensation values
3. **ADAPTIVE**: Adjusts compensation based on error magnitude
4. **CONSERVATIVE**: Uses 50% compensation to avoid overshooting

## Testing

The tool includes comprehensive pytest tests with mock support:

```bash
cd bias_tuner
python -m pytest tests/ -v
```

All tests use real experimental data from test_resources/ to ensure accuracy.

## Workflow

1. **Baseline Measurement**: Disable compensation and measure initial errors
2. **Layer 1 Compensation**: Apply compensation to layer 1 based on baseline
3. **Re-measurement**: Run inference and analysis with layer 1 compensated
4. **Layer 2 Compensation**: Calculate layer 2 compensation from new measurements
5. **Continue**: Repeat for each subsequent layer
6. **Report Generation**: Create comprehensive JSON report with results

## Key Findings

- SVF layer (layer 0) does not support bias compensation
- Dense layers 1-3 are compensatable with good results
- Output layer compensation can cause instability
- Sequential compensation accounts for layer coupling effects
