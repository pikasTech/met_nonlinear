## 1. 原始设计理念分析

### 1.1 核心设计原则

偏置调谐器基于**层序递进补偿**（Sequential Layer Compensation）的设计理念：

```python
# tuner.py:189-235 - tune_sequential()方法体现的设计哲学
def tune_sequential(self, layer_order, scale_factors):
    """按指定顺序对层进行序列微调"""
    for layer_idx in layer_order:
        # 1. 单层补偿计算
        result = self.tune_single_layer(layer_idx, scale)
        # 2. 配置状态保持
        # 3. 继续下一层
```

### 1.2 算法架构

**三阶段处理流程**:

1. **基线测量** (`run_baseline_measurement()`)
   - 禁用所有补偿
   - 获取原始偏置误差矩阵
   - 建立性能基准

2. **单层微调** (`tune_single_layer()`)
   - 读取当前层误差
   - 计算补偿值
   - 应用并保持补偿状态
   - 验证改善效果

3. **状态累积** (`current_compensation`)
   - 维护所有已补偿层的状态
   - 确保多层补偿的叠加效应
   - 支持回滚和重置

### 1.3 补偿策略

系统实现了**4种补偿策略**:

```python
class CompensationStrategy(Enum):
    SAME_PHASE = "same_phase"        # 同相100%补偿（主要策略）
    SCALED = "scaled"                # 缩放补偿
    ADAPTIVE = "adaptive"            # 自适应补偿
    CONSERVATIVE = "conservative"    # 保守50%补偿
```

**同相补偿**是核心策略，虽然违反直觉，但经实验验证有效：
```python
# compensator.py:75-95
def _same_phase_compensation(self, bias_errors, scale_factor=1.0):
    """同相补偿：使用相同符号和幅值进行补偿
    这与直觉相反，但符合实验结果"""
    compensation = [error * scale_factor for error in bias_errors]
```
