## 一、Wiener模型核心理论

### 1.1 核心结构

```
Wiener模型基本结构:
┌─────────────┐    ┌─────────────┐
│ 线性动态G(z) │ → │ 非线性f(·)  │
└─────────────┘    └─────────────┘

Wiener-Hammerstein (串联结构):
┌─────────────┐    ┌─────────────┐    ┌─────────────┐
│ G1(z)      │ → │  f(·)       │ → │ G2(z)       │
└─────────────┘    └─────────────┘    └─────────────┘

深度Wiener (SSM等价):
多层堆叠: G1(z) → f1(·) → G2(z) → f2(·) → ... → Gn(z)
```

### 1.2 关键公式

| 模型 | 结构表达式 |
|------|-----------|
| **Wiener** | y(t) = f(G(z)u(t)) |
| **Hammerstein** | y(t) = G(z)f(u(t)) |
| **Wiener-Hammerstein** | y(t) = G2(z)f(G1(z)u(t)) |
| **随机Wiener** | x(t+1) = Ax(t)+Bu(t)+w(t), y(t) = f(Cx(t))+v(t) |
| **深度Wiener (SSM)** | h_t = Ah_{t-1}+Bx_t, y_t = f(h_t) |

### 1.3 核心理论支撑

| 论文 | 核心贡献 | 支撑度 |
|------|----------|--------|
| Schoukens 2009 | Wiener-Hammerstein基准：G1(z)→f(·)→G2(z) | ✅ 强 |
| Haber 1990 | 块结构模型统一处理 | ✅ 强 |
| Bai 2010 | f(x)=Σc_jφ_j(x)正交基函数 | ✅ 强 |
| Bonassi 2023 | SSM≡深度Wiener模型 | ✅ 强(理论证明) |
| Cruz 2025 | SS-KAN：线性SS + KAN非线性 | ✅ 强 |
| Willemstein 2024 | 软鞋垫RMS 9.3%, R²=0.85 | ✅ 强(工程验证) |
| Wahlberg 2015 | 随机Wiener间接推理 | ✅ 中 |

### 1.4 与MET论文的关联

- **线性部分G(z)**: 对应MET传感器的递归/动态特性（热传导延迟、扩散过程）
- **非线性部分f(·)**: 对应传感器的静态非线性增益特性
- **Wiener-Hammerstein**: 考虑输入/输出端的额外滤波效应

## 二、KAN网络核心理论

### 2.1 Kolmogorov-Arnold定理基础

**定理内容**: 任意连续多变量函数可分解为单变量连续函数和加法运算的有限组合

**对KAN设计的指导意义**:
- KAN将多变量函数分解的数学原理直接应用于网络架构
- MLP在节点上使用固定激活函数，KAN将其移至边（权重）上，变为可学习的单变量B样条函数
- 避免维度灾难问题

### 2.2 B样条基函数分析

| 特性 | ReLU | B样条 |
|------|------|-------|
| 支撑域 | 全局 | 局部紧凑 |
| 连续性 | 分段连续 | 连续可微 |
| 表达能力 | 需宽网络 | 窄网络即可 |
| 频谱特性 | 高频偏差大 | 频谱分布均匀 |

### 2.3 关键定理（Wang et al. ICLR 2025）

- **定理3.2**: KAN可表示ReLU MLP（KAN表达能力≥MLP）
- **频谱偏差**: KAN对低频的偏向性小于MLP，更容易学习高频成分
- **收敛速率**: O(n^(-2r/(2r+1))) 极小极大最优

### 2.4 核心理论支撑

| 论文 | 核心贡献 | 支撑度 |
|------|----------|--------|
| Liu 2024 | 首个KAN；B样条激活 | ✅ 强 |
| Wang ICLR 2025 | KAN频谱偏差<MLP；定理3.2 | ✅ 强 |
| Liu 2025 | 收敛速率O(n^(-2r/(2r+1))) | ✅ 强 |
| Southworth 2026 | KAN≡多通道power ReLU MLP | ✅ 强 |
| TKAN 2024 | R²@12步：TKAN=0.104, GRU=0.018 | ✅ 强 |
| Rather 2025 | GRU/LSTM-KAN混合优于基线 | ✅ 强 |
| Vaca-Rubio 2024 | 109k参数KAN优于329k参数MLP | ✅ 强 |
| Kui 2025 | TFKAN双分支频域建模 | ✅ 强 |

### 2.5 与Wiener模型的关联

- **Wiener模型**: 线性动态系统 + 静态非线性函数级联
- **KAN理论**: 多变量函数分解为单变量函数叠加
- **核心联系**: 两者都基于"复杂函数可分解为简单函数组合"的数学原理
- **MET论文架构**: 线性递归 + KAN非线性，符合SS-KAN框架

## 三、频域损失函数理论

### 3.1 AFMAE损失函数分析

**代码实现** (`af_mse_loss`):
```python
# 1. 分组：group_points=4000
# 2. 计算每组能量：power = Σ|y| (按group维度求和)
# 3. 对数变换：log(power + 1e-8)
# 4. 平方误差：loss = (log_true - log_pred)²
```

**数学定义**:
```
L_afmse = E[(log(P_true) - log(P_pred))²]
其中 P = Σ|y| per group
```

### 3.2 "双重惩罚"效应（Subich ICML 2025）

**MSE的问题**:
```
预测误差 = 去相关误差(相位) + 振幅误差(能量)
MSE同时惩罚两者 → 模型倾向平滑细尺度特征
```

**频域损失如何解决**:
- FreDF/OLMA: 频域直接监督，不同频率独立评估
- AFMAE: 能量对数差，不直接惩罚相位，只惩罚能量分布
- BSP: 分箱能量比，避免单一尺度误差主导

### 3.3 关键公式汇总

| 损失函数 | 公式 |
|----------|------|
| FreDF | L^α = α·|F(Ŷ)-F(Y)|₁ + (1-α)·MSE |
| AFMAE (MET) | E[(log Σ|y_true| - log Σ|y_pred|)²] |
| KFS | ℒ = α·(1/K)∑||ℱ{y~}_i - ℱ{y}_i|| + (1-α)·ℒ_MSE |
| BSP | (1/N_k)·Σ_c Σ_i (1 - (E^bin_u+ε)/(E^bin_v+ε))² |
| Focal Frequency | 自适应频率权重：w(f) ∝ 1/(|F_pred - F_true| + δ) |

### 3.4 核心理论支撑

| 论文 | 核心贡献 | 支撑度 |
|------|----------|--------|
| FreDF Wang ICLR 2025 | L^α = α·|F(Ŷ)-F(Y)|₁ + (1-α)·MSE；定理3.3 | ✅ 强 |
| OLMA Shi ICLR 2026 | 熵减定理：DFT降低边缘熵 | ✅ 强 |
| Subich ICML 2025 | "双重惩罚"效应；MSE过度平滑 | ✅ 强 |
| Jiang 2020 | 自适应频率聚焦损失 | ✅ 强 |
| KFS Wu 2025 | ℒ_F + ℒ_MSE联合损失 | ✅ 强 |
| PETSA ICML 2025 | 频域项保持周期性 | ✅ 强 |

### 3.5 与MET论文的匹配度

✅ **高度匹配**
- MET声称AFMAE是"频域幅频率损失"
- 代码实现通过分组能量对数差实现频域能量分布匹配
- 符合FreDF、OLMA、BSP、KFS等顶级会议论文的理论框架
