## 3. 核心理论框架

### 3.1 Wiener-KAN架构

```
Wiener-KAN 架构
├── Wiener 模型理论
│   ├── 线性动态系统 G1(z) → 非线性静态 f(·) → 线性动态 G2(z)
│   ├── 经典理论：Schoukens 2009, Haber 1990, Bai 2010
│   └── 深度学习：LSTM-Wiener (Li 2024), SS-KAN (Cruz 2025)
├── KAN 网络
│   ├── B样条激活函数 + LUT计算
│   ├── 核心理论：Liu 2024 KAN, Barron空间理论
│   └── 效率证据：KAN-FIF (94.8%参数减少), LUT-KAN (12x加速)
└── AFMAE 损失函数
    ├── 公式来源：FreDF (Wang 2025 ICLR)
    └── 理论支撑：OLMA (熵减), Subich (双重惩罚), KFS (Parseval)
```

### 3.2 AFMAE公式确认

```
L^α = α·|F(Ŷ)-F(Y)|₁ + (1-α)·MSE
```

其中：
- F(·) = FFT傅里叶变换
- |·|₁ = L1范数
- α = 0.5 (典型值)
