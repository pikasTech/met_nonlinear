## 论文1：BSP损失用于混沌系统

**参考**：Chakraborty等 2025 - arXiv:2502.00472
**raw_literature.md中的状态**：待定

### 核心贡献

本文介绍了**分箱频谱功率（BSP）损失**，这是一种新颖的频域损失函数，旨在减轻深度学习模型在混沌系统预测中的频谱偏差。关键洞察是传统MSE损失逐点操作，自然倾向于首先学习低频，导致神经网络在多尺度动力系统中错过细粒度结构。

### 关键方法/公式

**BSP损失定义**（来自论文的方程7）：
```
L_BSP(u,v) = (1/N_k) * Σ_{c=1}^C Σ_{i=1}^{N_k} (1 - (E^bin_u(c,i) + ε)/(E^bin_v(c,i) + ε))²
```

其中：
- `E^bin_u(c,i)` = 通道c、箱子i的预测分箱频谱能量
- `E^bin_v(c,i)` = 通道c、箱子i的真实值分箱频谱能量
- `N_k` = 波数箱数量
- `ε` = 避免除零问题的小常数

**与多步展期损失结合**（方程8）：
```
L_R* = E_j[ Σ_{t=1}^m ||γ(t)(F_φ^t(x_j) - G^t(x_j))||² + μ * L_BSP^{(j,t)} ]
```

**算法总结**：
1. 将预测和真实场变换到傅里叶域
2. 计算每模态能量：E = (1/2)|û|²
3. 按波数幅度将能量分箱
4. 计算每个箱的能量比率的平方相对误差
5. 按不同频带的λ_i加权（通常λ_i = 1）

### 引文

> "BSP损失是一种频域损失函数，自适应地权衡预测大小尺度和更小尺度上的误差。与关注逐点拟合的传统损失不同，我们的BSP损失明确惩罚不同尺度上能量分布的偏差，促进稳定和物理一致的预测。"

### 与AFMAE的关系

**直接相关性：高**

根据raw_literature.md备注（第145行）：
> "BSP损失（Chakraborty 2025）- **最相似概念**：自适应频域bin权重+MAE"

这正是AFMAE概念的精确描述。BSP损失：
- 使用**自适应频带权重**（λ_i）
- 使用**MAE风格的平方相对误差**公式
- 明确针对**频带间的能量分布**
- 架构无关，可与现有模型结合

BSP损失是所有分析论文中与AFMAE**方法论上最相似**的。

## 论文2：Fre-CW（频域Carlini-Wagner攻击）

**参考**：Feng等 2025 - arXiv:2508.08955
**raw_literature.md中的状态**：已排除（P0高）

### 核心贡献

本文提出**Fre-CW**，这是一种利用频域特征的时间序列预测模型对抗攻击算法。它将Carlini-Wagner（C&W）攻击从分类领域适应到预测领域，使用时域和频域损失来生成欺骗预测模型的扰动。

### 关键方法

1. **C&W攻击的适应**：最初为分类设计，适应于序列到序列预测
2. **双域损失优化**：将传统时域MSE与频域损失结合
3. **攻击目标**：生成导致模型产生错误预测的扰动

**损失结构**（来自摘要）：
```
使用时域和频域损失优化对抗样本
```

### 引文

> "基于Transformer的模型在时间序列预测方面取得了重大进展。然而，深度学习模型的一个关键局限性是它们容易受到对抗攻击，这在时间序列预测的背景下尚未得到充分研究。"

> "据我们所知，没有相关研究将频率信息用于时间序列对抗攻击。"

### 与AFMAE的关系

**直接相关性：无/负面**

关键区别：
- **AFMAE目标**：通过更好的频域感知训练提高预测精度
- **Fre-CW目标**：通过对抗扰动攻击/降低模型预测
- **方法方向**：相反 - AFMAE增强预测，Fre-CW攻击预测

这是一篇**对抗攻击论文**，不是训练方法论文。

### 排除确认

**已在raw_literature.md（第37行）标记为排除** - 确认正确。

**排除原因**：
1. **目标不兼容**：AFMAE用于改进模型训练/预测；Fre-CW用于攻击/降低模型
2. **应用领域**：安全/对抗ML，不是测量/估计改进
3. **无训练益处**：虽然使用频域损失，但目的是破坏性的，不是建设性的

### 建议

**排除**（已标记）

**理由**：
1. 攻击聚焦方法论 - 与AFMAE改进目标相反
2. 安全/对抗ML论文，不是测量/传感器建模
3. 对MET系统辨识或传感器漂移补偿没有建设性应用
4. 仅与模型脆弱性评估相关
