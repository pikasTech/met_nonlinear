## 论文1：Hsu, Chou, Kuo (2017)

### 基本信息
- **标题**："使用Wiener型循环神经网络对MEMS陀螺仪进行漂移建模与补偿"
- **期刊**：IEEE International Inertial Sensors and Systems
- **链接**：https://ieeexplore.ieee.org/document/8270067
- **状态**：付费内容 - IEEE Xplore（18次引用）

### 核心贡献
- 用于MEMS陀螺仪漂移建模的Wiener型循环神经网络（WRNN）
- 架构：动态线性模型 + 静态非线性模型（Wiener结构）

### 关键方法
```
WRNN结构：
u(t) → [线性动态模型 G(z)] → x(t) → [静态非线性 f(·)] → y(t)

- 线性部分：ARMA型动态模型
- 非线性部分：基于神经网络的静态非线性
- 训练：递归循环学习算法
```

### 关键发现
- WRNN有效建模MEMS陀螺仪的固有漂移
- 网络架构的自构造机制
- 补偿提高了陀螺仪精度

### 与MET非线性的相关性
- **中等** - Wiener模型在MEMS传感器中的直接应用
- 与MET传感器类似：线性动力学 + 非线性特性
- 可为Wiener-KAN提供方法论参考

## 论文5：Li, Zhou, Liu (2024)

### 基本信息
- **标题**："Hammerstein-Wiener参数估计及其在锂离子电池中的应用"
- **期刊**：Journal of Energy Storage
- **链接**：https://www.sciencedirect.com/science/article/pii/S2352467724003937
- **状态**：付费内容 - ScienceDirect（11次引用）

### 核心贡献
- 使用混合信号对Hammerstein-Wiener系统进行参数估计
- 神经网络模糊网络（NFN）用于静态非线性块
- FIR模型用于线性动态块
- 应用于锂离子电池SOH估计

### 关键方法
```
Hammerstein-Wiener结构：
静态非线性 f(·) → 线性块 G(z) → 静态非线性 h(·)

- 两个NFN块建模输入/输出非线性
- FIR模型用于线性动态
- 多创新技术提高精度
- 分离信号 + 随机信号用于激励
```

### 关键公式
- 输出NFN模型参数：聚类算法 + 相关性分析
- FIR参数：基于相关函数的最小二乘法
- 输入NFN的多创新最小二乘法

### 关键发现
- 带NFN的Hammerstein-Wiener有效建模电池动态
- 锂离子电池容量精确预测
- 混合信号激励改善参数估计

### 与MET非线性的相关性
- **高** - Hammerstein-Wiener结构验证
- NFN替换静态非线性 → KAN可类似替换
- 为块结构非线性模型提供基准
- 电池应用展示了实用性

- **原因**：摘要确认方法论；详细公式需要完整论文
