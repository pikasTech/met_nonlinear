# GAP7: 前馈补偿利用非线性区

## GAP定义

**核心声称**: 前馈补偿的方法利用了非线性区，而不是排除了非线性区，这样可以提升更大的量程

**具体描述**: 传统反馈补偿方法试图排除或抑制非线性区域，而前馈补偿方法可以主动利用非线性区域，从而实现更大的量程提升。
