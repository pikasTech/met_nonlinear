需要考虑一个问题，那就是是否要让推理输出的数据保留有 input 通道，如果保留了 input 通道，那么单个 wave 文件就可以进行频率响应分析，否则就必须要有一个 input wave 文件和一个 output wave文件。但是如果保留了 input 通道，那么多层推理的时候，应该保留哪个 input ？逐级保留会导致数据过于复杂，只保留最前面的 input 又不能分析层间的怕你率响应，且对于层间的透明性也有负面作用。如果 input wave 和 output wave 相互分离，那么进行频响分析的时候就必须要使用两个 wave 文件，但是好处是可以用同一个框架分析任意层之间的频响。

最终我决定不保留输入数据，将 input 波形和 output 波形分离为两个 wave，并在计算频率响应时需要传入两个 wave 文件。

> • 做“精密绝对值/整流”时，把二极管放在运放反馈环内（经典图：运放 + 二极管 + 电阻），此时二极管的正向压降被环路增益压缩到 μV 级，唯一真正影响精度的是反向漏电流，因此 BAS116/PAD5 的低漏电优势被放大。
> • 如果需要亚微伏级漂移，可把两只二极管背靠背并联，抵消温度系数，再用低 Vos 运放（如 ADA4522、LTC2050、OPA2188 等）。
>
> 结论
> 低频高精度场合，直接买 BAS116（或 PAD5）即可，它正是 ADI 官方电路里反复出现的“精密整流二极管”。

出于上面考虑，二极管选 BAS116

把“影响低频精密整流精度”的关键指标逐项拆开，把 1N4148 与 BAS116 的典型数据摆在一起，就能看出差距：

| 关键参数                    | 1N4148 典型值       | BAS116 典型值                  | 精度影响                                                     | 备注 |
| --------------------------- | ------------------- | ------------------------------ | ------------------------------------------------------------ | ---- |
| 反向漏电流 I<sub>R</sub>    | 25 nA @ 20 V，25 °C | 3 pA @ 20 V，25 °C             | 漏电流直接叠加到被整流信号 → 成为整流“零偏误差”。BAS116 比 1N4148 低 3~4 个数量级，可把 16 bit 系统的 LSB 误差从 mV 级压到 μV 级。 |      |
| 正向压降 V<sub>F</sub>      | 1.0 V @ 10 mA       | 0.6 V @ 1 mA（0.75 V @ 10 mA） | 在“运放反馈整流”架构里，V<sub>F</sub> 会被环路增益压缩，最终决定残余失调的是 dV<sub>F</sub>/dT 与 V<sub>F</sub> 离散性。BAS116 温度系数更低，离散性更小。 |      |
| 温度系数 ΔV<sub>F</sub>/ΔT  | ≈ –2.0 mV/°C        | ≈ –1.8 mV/°C                   | 温漂乘上闭环增益后，变成整流输出漂移；系数越小，温漂越小。   |      |
| 反向恢复时间 t<sub>rr</sub> | 4 ns                | 4 ns（同量级）                 | 低频整流对 t<sub>rr</sub> 不敏感，可忽略。                   |      |
| 结电容 C<sub>0</sub>        | 4 pF                | 2 pF                           | 只在 >100 kHz 才明显影响；低频整流可忽略。                   |      |

在低频、高精度整流场景里，**反向漏电流是头号敌人**。
1N4148 的 25 nA 已能把 16-bit、±5 V 系统的 LSB（≈76 µV）淹没；BAS116 只有 3 pA，可以安全地做到微伏级精度。正向压降及其温漂次之，BAS116 同样占优。因此，**若目标是 12-bit 以上、μV~mV 级误差的精密整流，BAS116 明显优于 1N4148**
