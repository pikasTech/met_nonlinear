## 第二稿声称的贡献 3月29修订

现在已经调研了相当多的文献，下一步的任务是明确哪些论文可以作为 GAP 的支撑

第二稿的声称贡献草稿：

- MET 非线性问题
  - 机理分析 - 最近补充了机理分析，这个是新的
    - 这是一个主要 GAP 应当引用一些电化学地震检波器的频响漂移的论文 (GAP1)
      - 尤其是要引用对温度漂移的研究，支撑从温度漂移到非线性漂移的 GAP
  - 实验测量 - 实验测量，这个是新的
    - 这个 GAP 也要引用一些电化学地震检波器的研究来支撑
      - 非频率漂移研究：这部分主要是支撑线性度的测量范围，都偏窄（已有的论文里面已经有充分支撑） (GAP2)
      - 频率漂移的研究：说明频率漂移研究的影响因素已有的是温度，缺乏震级因素的影响 (GAP3)
    - 展开说明测量的过程，信号的频率属性，数据集的划分等
    - 参考其他发布新数据集的论文
  - 建模模拟 - 用 Wiener 并联系统做了建模，这个是新的
    - GAP 在于建模如何与补偿方法联系起来'
    - 如果建模和补偿都用 Wiener 
    - Wiener 和 FRIKAN 是能联系的，线性部分对应 IIR，非线性部分对应 KAN
    - 或者将 FRIKAN 改名 Wiener-KAN？这样线性部分直接就一一对应了，非线性部分对应 KAN 了，建模和补偿方法的联系就更紧密了
    - 统一用 Wiener-KAN 来完成建模和补偿？这样就通了
    - 这个 GAP 主要是说，以前没有人做过电化学地震检波器关于震级的频响漂移的建模
      - 非频率漂移研究：推导了电化学地震检波器的线性模型，而没有非线性模型 (GAP4)
      - 频率漂移的研究：建模了温度因素，没有建模震级因素对频率漂移的影响 (GAP5)

- 补偿方法
  - 神经网络前馈补偿架构 - 其实非线性问题也有用神经网络的
    - 是否和反馈补偿做比较？ - 应该是可以比较的，因为震级明显大了
    - 是否和传统的非线性神经网络补偿做比较 - 可以的，MEASUREMENT 本身就有发这个温度非线性补偿的论文
      - 以往的方法通过力反馈来抑制非线性，这样限制了最大量程的提升，而前馈补偿方法没有这个限制，可以提升更大的量程(GAP6)
      - 前馈补偿的方法利用了非线性区，而不是排除了非线性区，这样可以提升更大的量程(GAP7)
  - Wiener-KAN 模型
    - 组件提出
      - 声称 KAN 相对于 Wiener 传统非线性函数的可训练性改进
      - 声称 KAN 的 LUT 计算效率加速
      - 改为声称 Wiener 的线性部分
        - 声称 IIR 的计算参数少于 1D-CNN/RNN/LSTM，计算效率更高
        - 线性部分的高计算效率是很好声称的
    - 性能声称
      - 计算效率改进 
        - KAN 做 LUT 或许有实质的计算效率改进（相对 LSTM、GRN、TRANSFORMER）
        - IIR 相对于 1D-CNN/RNN/LSTM 有实质的计算效率改进
    - GAP
      - 频率无关的非线性补偿方法：作比较，支撑频率相关的补偿能力，补偿精度(GAP8)
      - 频率相关的非线性补偿方法: 做比较，来支撑计算效率的提升 (GAP9)
  - 训练方法
    - AFMAE 损失函数 - 需要做消融实验，但是大概率是有提升的
      - 要和纯 MAE 做比较，来支撑 AFMAE 的改进 (GAP10)
      - 与其他频率相关的损失函数做比较，来支撑 AFMAE 的效率改进和简单性（直接计算能量，无需FFT）(GAP11)
