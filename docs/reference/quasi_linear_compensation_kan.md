# 准线性补偿视角下的 Wiener-KAN 结构可行性论证

本文档从准线性（Quasi-Linear）系统的物理特性出发，论证为什么基于“线性动态 + 静态非线性”的并联 Wiener 结构能够逼近复杂非线性系统的逆解，以及为什么 Kolmogorov-Arnold 网络（KAN）是解决该结构的通用求解途径。

## 1. 准线性特性与幅频耦合近似

电化学地震检波器（MET）在不同震级 $m$ 的正弦稳态激励下，表现出灵敏度和固有频率的连续漂移。由于在每个固定的窄带稳态工况下，系统输出仍以基频分量为主，其行为可以被抽象为一个随震级 $m$ 参数化变化的准线性传递函数 $H(s, m)$。

为了实现全量程的前馈补偿，理想的补偿器传递函数 $H_{\text{relative}}$ 应满足：
$$
H_{\text{relative}}(s, m) \approx \overset{\circ}{H}(s) / H(s, m)
$$
其中 $\overset{\circ}{H}(s)$ 为期望的理想线性目标响应。

直接推导高维物理场强耦合系统在任意 $m$ 下的全局 $H^{-1}(s, m)$ 是极为困难的。但在正向建模中，前人研究与本文的实验已证明：随幅值变化的 $H(s, m)$ 可以通过一组并联的局部动态来近似解耦，即：
$$
y(t, m) \approx \sum_{i} f_i\Big( \big(h_i * x\big)(t, m) \Big)
$$
在这个正向物理模型中：
*   **线性动态 $h_i(t)$**：代表不同震级区间下的局部线性频率响应，可由实测扫频数据直接辨识。
*   **非线性映射 $f_i(\cdot)$**：代表局部的幅度畸变与过渡权重。

这种基于“实测局部动态 + 经验试凑非线性”的并联 Wiener 建模方法，在正向上成功复现了幅频耦合曲线（例如固有频率和 $\valWienerParallelGainFreq$ Hz 灵敏度的轨迹），这**在理论上证明了将复杂时空非线性解耦为“多路线性滤波 + 静态非线性组合”这一结构是可行的**。

## 2. 逆向补偿的结构映射与试凑瓶颈

既然正向模型可以被分解，那么根据结构对称性，补偿器的准线性逆解也可以尝试采用类似的并联形式进行结构逼近：
$$
\hat{y}(t) \approx \sum_{j} f_{\text{comp}, j}\Big( \big(\overset{\rho}{h}_j * y\big)(t) \Big)
$$
在这个逆向补偿结构中：
1.  **逆滤波部分 $\overset{\rho}{h}_j(t)$**：可由相对频率响应 $\overset{\rho}{H}_j(s) = \overset{\circ}{H}(s) / H_j(s)$ 计算得到，属于已知先验。
2.  **逆向非线性补偿映射 $f_{\text{comp}, j}(\cdot)$**：对应于消除主支路残余误差与跨支路非线性交调所需的映射函数。

**求解瓶颈**：
在正向建模中，$f_i(\cdot)$ 可以通过观测稳态响应幅值、凭人工经验（例如多项式、双曲正切或带死区函数）进行试凑与拟合。然而，在逆向补偿中，$f_{\text{comp}, j}(\cdot)$ 必须处理宽频带复杂信号经过逆滤波器后产生的时域畸变与相邻支路交调。它不再是简单的标量幅值映射，而是一个复杂的高维空间非线性逆问题。**依靠经验试凑 $f_{\text{comp}, j}(\cdot)$ 变得完全不切实际，这是阻碍此类前馈结构走向通用的核心难点。**

## 3. KAN 激活函数：提供非线性映射的通用求解法

传统的多层感知机（MLP）依赖固定的激活函数（如 ReLU），只能通过海量权重矩阵隐式地逼近 $f_{\text{comp}}$，既难以收敛，又会破坏原本清晰的物理分解结构。

Wiener-KAN 的提出完美填补了这一结构与求解之间的鸿沟。Kolmogorov-Arnold 网络（KAN）的核心创新在于：**将可训练的一维样条函数（Splines）直接放置在网络边缘上作为激活函数**。

如果我们把并联补偿结构中的未知映射 $f_{\text{comp}, j}$ 直接视作 KAN 网络中需要学习的样条激活函数 $\phi(\cdot)$，那么：
1.  **物理结构的保留**：Wiener 线性层承担了 $\overset{\rho}{h}_j(t)$ 的角色，显式注入了逆滤波先验。
2.  **试凑法的终结**：KAN 边缘上的 B 样条系数通过端到端梯度反向传播自动更新。网络不再需要人类专家去猜测补偿函数的具体形式（多项式还是指数形式），而是通过数据驱动，自适应地将最佳的逆向融合映射 $f_{\text{comp}}$ 学习出来。

**结论**：
从准线性系统视角来看，Wiener 结构确立了非线性补偿的可行解空间骨架，而 KAN 则通过可训练样条函数彻底打通了 $f_{\text{comp}}$ 的通用求解路径。两者结合，使得大震级幅频耦合系统的实时前馈补偿从“经验试凑”走向了“自适应最优求解”。

## 4. 物理先验的注入与权重冻结的必要性

在具体的 Wiener-KAN 工程实现中，逆滤波器 $\overset{\rho}{h}_j(t)$ 被作为前置的线性层（如通过 IIR 算子或线性循环单元）注入到网络中。从准线性视角的整体结构来看，无论采取何种工程实现方式，其配套的“权重冻结（Frozen）”策略都具有深刻的科学必要性。

### 为什么必须冻结线性层的权重？
在 Wiener-KAN 的训练中，通过相对频率响应初始化的线性前置层必须被冻结，禁止参与梯度下降。这恰恰是“准线性补偿视角”的核心要求：

1.  **坚守物理先验的纯洁性**：根据前述第 2 节的分析，逆向并联结构的线性动态基底 $\overset{\rho}{h}_j(t)$ 是**已知的物理事实**（由传感器客观的频率响应直接推导而来）。如果放开这部分权重参与网络训练，模型极易陷入过拟合的局部最优，导致它为了强行迎合时域上的某类逐点误差而改变原有的物理传递函数，最终表现为在“频率响应漂移（Freq Drift）”等宏观频域指标上的全面崩溃（正文中的消融实验已严格证实了这一点）。
2.  **职责的强制解耦**：冻结机制强制实施了严格的分工——让线性前置层安心充当绝对可靠的准线性时空特征提取器，提供符合物理基底的无偏中间信号 $z_j(t)$；而把处理跨支路交调串扰（极难建模的高维非线性残差）的所有压力，全部干净地推给具备极致高维拟合能力的多层 KAN 网络去承担。

**总结**：在端到端补偿网络中注入并冻结物理先验，是为了在深层网络中**强制守护准线性解耦理论的纯粹性**，从机制上杜绝纯数据驱动模型对客观物理规律的“污染”，这也是 Wiener-KAN 能够在宽频带上始终保持频率响应稳定的根本原因。
