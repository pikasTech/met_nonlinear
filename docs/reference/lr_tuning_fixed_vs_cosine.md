# Learning Rate 调参实验报告

## 实验目标

探索 FRIKAN 模型在固定学习率下能否达到余弦衰减（Cosine Decay）相同或更好的验证损失（val_loss）。

目标值：val_loss < 0.02125（基线使用余弦衰减）

## 实验配置

- **模型架构**: FRIKAN, H_UNITS=8, INNER_KAN_UNITS=6, INNER_KAN_LAYERS=6
- **训练配置**: epoch=1000, step_per_epoch=5, use_power_loss=True, use_best_val_weights=True
- **数据集**: 8000 points, sample_rate=2000
- **基线**: use_auto_lr=True, auto_lr_decay_steps=500, lr_max=0.002, lr_min=1.62e-5, lr_mean=0.001053

## 实验结果

| 项目名 | 学习率 | min_val_loss | 与基线差值 |
|--------|--------|--------------|-------------|
| 基线（余弦衰减） | 0.002→1.6e-5 | **0.02125** | - |
| FRIKANh8u6l6_e1k_lr5e4 | 0.0005 | 0.02354 | +0.00229 |
| FRIKANh8u6l6_e1k_lr1e3 | 0.001 | 0.02436 | +0.00311 |
| FRIKANh8u6l6_e1k_lr7e4 | 0.0007 | 0.02439 | +0.00314 |
| FRIKANh8u6l6_e1k_lr4e4 | 0.0004 | 0.02615 | +0.00490 |
| FRIKANh8u6l6_e1k_lr6e4 | 0.0006 | 0.02628 | +0.00503 |
| FRIKANh8u6l6_e1k_lr2e4 | 0.0002 | 0.02782 | +0.00657 |

## 问题分析

### 1. 固定 LR 无法复现余弦衰减效果

余弦衰减的优势在于：
- **前期高 LR (0.002)**: 快速探索参数空间
- **后期低 LR (1.6e-5)**: 精细收敛到最优解

固定 LR 的问题：
- **LR 过大 (0.001-0.002)**: 过冲，无法收敛到更低的损失
- **LR 过小 (0.0002-0.0004)**: 学习缓慢，在 1000 epoch 内无法充分收敛
- **LR=0.0005**: 最佳固定 LR，但仍比余弦衰减差约 10%

### 2. 最优固定 LR 约 0.0005

在固定 LR 设置下，lr=0.0005 取得最佳 val_loss=0.02354。LR 在 0.0005 附近形成了一个局部最优区间。

### 3. 过小的 LR 导致收敛缓慢

lr=0.0002 训练 1000 epoch 后 val_loss 仍达 0.02782，明显差于其他配置。

## 结论

1. **余弦衰减策略对于本任务至关重要**，固定 LR 难以达到相同效果
2. 如果必须使用固定 LR，建议从 **lr=0.0005** 开始
3. 固定 LR 与余弦衰减的差距约 **10%** (0.00229/0.02125)

## 建议

如需使用固定 LR 方案，考虑：
- 增加训练 epoch 数
- 使用两阶段训练：先高 LR 快速探索，再低 LR 精细收敛
- 尝试其他衰减策略（如指数衰减、阶梯衰减）
