# 并联 Wiener 模型到 Wiener-KAN 的逆解与补偿推导

本文档记录从传统三支路并联 Wiener 等效模型推导至 Wiener-KAN 神经网络补偿架构的核心数学动机，主要论证为何在具有强物理耦合的 MET 系统中，传统的“解析求逆”方法会失效，从而必须采用基于深度学习的端到端补偿。

## 1. 物理模型抽象与前馈补偿的目标

根据 MET 在大震级下的幅频耦合漂移现象，系统输出 $y(t)$ 可由三支路并联 Wiener 等效模型精确表征：
$$
y(t) = \sum_{i=1}^{3} f_i\left((h_i * x)(t)\right)
$$
其中：
*   $x(t)$：输入振动信号（加速度或速度）。
*   $h_i(t)$：第 $i$ 支路的局部线性动态核（对应不同震级范围下频率响应的迁移）。
*   $f_i(\cdot)$：第 $i$ 支路的静态非线性映射。

在理想情况下，传感器输出应为完全线性的 $\overset{\circ}{y}(t) = (\overset{\circ}{h} * x)(t)$。为消除由局部动态 $h_i$ 与静态非线性 $f_i$ 引入的幅频耦合畸变，前馈补偿器的目标是接收未补偿的 $y(t)$，并输出逼近 $\overset{\circ}{y}(t)$ 的校正结果。

## 2. 单支路 Wiener 系统的解析求逆（The "Easy" Inverse）

为了理解并联模型的复杂性，我们首先回顾如果系统退化为最简单的单支路 Wiener 模型时，解析求逆是如何生效的。

假设系统仅存在单一全局动态 $h$ 和单一静态非线性 $f$，输出为：
$$
y(t) = f\big((h * x)(t)\big)
$$

对其进行完全解析求逆非常直接，只需按照相反的顺序应用逆映射和逆滤波即可。
1. **静态非线性求逆**：首先通过逆函数 $f^{-1}(\cdot)$ 消除非线性畸变，得到中间信号 $v(t)$：
   $$
   v(t) = f^{-1}\big(y(t)\big) = (h * x)(t)
   $$
2. **线性滤波补偿**：接着，通过相对传递函数滤波器 $\overset{\rho}{h} = \overset{\circ}{h} * h^{-1}$ 恢复理想响应：
   $$
   \hat{y}(t) = (\overset{\rho}{h} * v)(t) = \big(\overset{\circ}{h} * h^{-1} * h * x\big)(t) = (\overset{\circ}{h} * x)(t) = \overset{\circ}{y}(t)
   $$

在这种单支路情况下，解析前馈补偿可以完美消除非线性并恢复目标频率响应，无任何余项残留。

## 3. 并联多支路模型面临的数学困境

然而，MET 真实的幅频耦合系统是多支路并联的。为了消除多支路系统的非线性，如果我们尝试套用上述成功的 Hammerstein 补偿器思路（即：先进行非线性求逆，再进行线性滤波），就会遇到根本性的数学阻碍。

假设我们对多支路混合信号 $y(t)$ 直接应用第 $j$ 支路的非线性逆映射 $f_j^{-1}(\cdot)$，试图以此分离出第 $j$ 支路的线性分量。此时中间信号 $v_j(t)$ 为：
$$
v_j(t) = f_j^{-1}\big(y(t)\big) = f_j^{-1}\left( \sum_{i=1}^{3} f_i\left((h_i * x)(t)\right) \right)
$$
展开求和项，分离出主支路与跨支路串扰：
$$
v_j(t) = f_j^{-1}\left( f_j(h_j * x) + \sum_{i \neq j} f_i(h_i * x) \right)
$$
此时，解析求逆方法暴露了导致其失效的核心数学盲区：**非线性逆算子的不可分配性（Non-distributivity of Nonlinear Inverse）**。

由于 $f_j^{-1}(\cdot)$ 是非线性函数，它不满足加法分配律，即 $f_j^{-1}(A + B) \neq f_j^{-1}(A) + f_j^{-1}(B)$。因此：
$$
v_j(t) \neq (h_j * x)(t) + f_j^{-1}\left( \sum_{i \neq j} f_i(h_i * x) \right)
$$
这意味着，跨支路串扰项 $\sum_{i \neq j} f_i$ 在经过非线性映射 $f_j^{-1}(\cdot)$ 时，会与主支路信号 $f_j(h_j * x)$ 发生严重的**非线性交调（Nonlinear Intermodulation / Mixing）**。主支路的线性信号 $(h_j * x)(t)$ 被彻底“污染”，即使后续再串联理想的逆滤波器 $\overset{\rho}{h}_j$，也无法从中提取并恢复出纯净的理想响应 $\overset{\circ}{h} * x$。

**结论**：
1. **Wiener 补偿器（先线性，后非线性）** 会因为算子的不可交换性（$\overset{\rho}{h} * f \neq f * \overset{\rho}{h}$）而导致主支路产生随频率变化的记忆误差。
2. **Hammerstein 补偿器（先非线性，后线性）** 虽然能完美抵消单支路 Wiener 模型，但在面对多支路并联系统的求和输出时，会因为非线性算子内部的不可分配性产生极其复杂的交叉频移与交调串扰。

因此，无论如何组合一维的解析逆映射和线性滤波，多通道并联 Wiener 系统的逆解在数学上都变得极端复杂且难以解析化简。

## 4. 神经网络化：Wiener-KAN 的架构动机

正是基于上述数学盲区，必须将前馈补偿结构彻底神经网络化。

1.  **Wiener 线性层的降维提取**：利用相对频率响应初始化的多核 Wiener 线性层同时提取各工作点下的频响特征 $\{z_j(t)\}$。该层不寻求直接“求逆”恢复理想信号，而是作为一组物理特征提取器，保留各支路的局部物理先验。
2.  **多层 KAN 消除动态余项**：抛弃独立的一维静态求逆，转而采用多层、多输入的 KAN（Kolmogorov-Arnold Network）。多层 KAN 能够接收多维特征 $\{z_1(t), z_2(t), z_3(t)\}$，依靠深层样条的高维非线性表达能力，以端到端学习的方式同时拟合并消除：
    *   算子不可交换性引入的记忆非线性误差。
    *   跨支路耦合带来的交调串扰。

Wiener-KAN 架构通过“物理驱动特征提取 + 数据驱动残差消除”的思路，实现了从单维静态映射向高维动态插值的转变，从而在数学上补齐了传统解析模型无法跨越的非线性宽带补偿鸿沟。
