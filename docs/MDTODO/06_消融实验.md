## 总任务

- 支持一个新的 ex_project 的命令行参数
  - 这个 ex_project 可以在 config.json 中指定多个 project 进行对比
  - 对比的指标可以在 config.json 中进行配置

- 示例的对比 ex_project
  - 示例的对比 ex_project - `ex_projects\compare\mae_vs_afmae`
  - 参与对比的 project - `projects\00_MAE_VS_AFMAE`
  - 参与对比的指标
    - 固有频率的漂移最大绝对值（百分比误差）
    - 100Hz 处的灵敏度漂移最大绝对值（百分比误差）
    - 1 - R^2 的平均线性度（百分比、对各个频率平均）

## R3 [completed] 

按照以下要求对 `projects\05_1DCNN` 和 `projects\06_TCN` 进行参数调优：
1. 每次只训练一个 project，避免系统资源过载导致的训练失败。
2. 调优只允许修改 `learning_rate` 和 `model_subcfg` 中的参数，其他训练配置保持不变。
3. 调整参数时需要先copy config.json 到新 proejct 目录然后修改参数，再训练。
4. 不要一次性生成批量的待训练 project，而是每次跑完之后分析训练结果指标，再决定下一步的参数调整方向。
5. 指标要求：
  - Freq Drift (Hz)∈(2.34Hz, 5Hz)
  - Sens Drift (%)∈(9.09%, 15%)
  - Linearity (%)∈(0.89%, 2.0%)
6. 自主调参，达到指标要求前禁止停下。禁止用 `question` 或者其他方式寻求帮助，必须自己分析训练结果并决定下一步调参方向。禁止使用 `plan mode`。

完成任务后将详细报告写入[R3](./details/06_消融实验/20260413_170317_Task_Report.md)。
