## 重点覆盖的功能

### exam_class.py 重点覆盖

1. **TimeSeries 高级方法**
   - `filter()`: 低通、高通、带通滤波器
   - `resample()`: 重采样（支持抗混叠滤波）
   - `clip()`: 按时间或索引裁剪
   - `concatenate()`: 连接多个TimeSeries
   - `remove_dc()`: 移除直流偏置
   - `invert()`: 信号反转
   - `flip()`: 信号翻转
   - `normalize()`: 归一化
   - `map()`: 映射函数
   - `limit()`: 限制范围

2. **二进制IO操作**
   - `dumptobinary()` / `loadfrombinary()`: 单通道二进制
   - `dump_multichannel_to_binary()` / `load_multichannel_from_binary()`: 多通道二进制

3. **System 类方法**
   - `fromGainPhase()` / `fromNumber()` / `fromSymbol()`: 多种创建方式
   - `toabs()` / `tophase()` / `tonumber()`: 获取系统属性
   - `parallel()`: 阻抗并联计算
   - `clone()`: 克隆系统

### waveprocessor.py 重点覆盖

1. **save/load 完整流程**
   - 使用内部状态保存
   - 多次保存加载循环
   - 状态更新验证

2. **边界情况**
   - 零振幅
   - 极短/极长信号
   - 单周期
   - 带噪声/直流偏置的信号

3. **多通道支持**
   - 双通道数据分析
   - 自定义通道索引
