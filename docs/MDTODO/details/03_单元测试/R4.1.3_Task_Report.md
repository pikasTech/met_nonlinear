## 使用说明

```bash

conda.bat run --no-capture-output -n tf26 python -m pytest src/tests -n 2

# 指定每个worker运行的测试数（Windows必需）
conda.bat run --no-capture-output -n tf26 python -m pytest src/tests -n 2 --tests-per-worker=1

# 使用所有可用CPU核心
conda.bat run --no-capture-output -n tf26 python -m pytest src/tests -n auto
```
