
**关键发现**: R1.8.3 报告中声称的 ~75% 覆盖率**从未被实际测量到**。coverage 工具未正确收集任何覆盖率数据，原因是：

3. **conftest.py 配置缺失**: 未正确配置 coverage 的 source 参数
