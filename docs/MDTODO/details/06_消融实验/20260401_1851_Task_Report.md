# R1 消融实验方案

**任务**: 支持一个新的 ex_project 命令行参数，用于多项目消融实验对比  
**日期**: 2026-04-01  
**状态**: 方案待评审

---

## 1. 任务目标

### 1.1 功能需求

1. **新增 ex_project 命令行参数**: 支持 `ablation-study` 类型的任务
2. **多项目对比**: 在 `config.json` 中指定多个 project 进行对比
3. **可配置指标**: 对比指标可在 `config.json` 中配置

### 1.2 对比指标

| 指标 | 描述 | 计算方式 |
|------|------|----------|
| 固有频率漂移 | 固有频率的最大绝对漂移（百分比误差） | `max(\|f_n_max - f_n_median\|, \|f_n_median - f_n_min\|) / f_n_median * 100` |
| 灵敏度漂移 | 100Hz 处灵敏度的最大绝对漂移（百分比误差） | `max(\|S_max - S_median\|, \|S_median - S_min\|) / S_median * 100` |
| 线性度 | 1 - R² 的平均值（百分比） | `mean(1 - r_squared) * 100` |

### 2.2 数据来源

各项目的 `linear_response.json` 包含：

```json
{
  "gains_origin": [[...], ...],      // 各幅度下的增益列表
  "gains_comped": [[...], ...],       // 补偿后增益
  "magnitudes": [...],                 // 幅度列表
  "frequencies": [...],                // 频率列表
  "fit_params_origin": [[A, B, C], ...],  // 原始系统拟合参数
  "fit_params_comped": [[A, B, C], ...]   // 补偿后拟合参数
}
```

### 2.3 指标计算逻辑

**固有频率 (fn)** - 来自 `fit_params`:
```python
# ProjectResult._calculate_parameters()
wn = sqrt(B)  # 角频率
fn = wn / (2 * pi)  # 固有频率
```

**灵敏度 (Sn)**:
```python
# ProjectResult.process_data()
Sn = A / (4 * pi * zeta * fn)
```

**灵敏度漂移计算** - 来自 `image_data_process.py`:
```python
def calculate_drift(metric):
    median = metric['median']
    return max(abs(metric['max'] - median), abs(median - metric['min']))
```

**线性度** - 来自 `linearity_by_frequency.json`:
```python
# 每个 frequency 有 r_squared_origin 和 r_squared_comped
linearity = mean(1 - r_squared) * 100
```

**复用组件**:

|------|------|------|
| `ProjectResult` | `figure_paper.py` | 数据加载和处理 |

| `linearity_by_frequency.json` | 项目 data 目录 | R² 线性度数据 |

**新增文件 `src/visualization/ablation_study.py`**:

```python
import json
import numpy as np
from visualization.figure_paper import ProjectResult
from visualization.image_data_process import calculate_drift


class AblationStudyAnalyzer:
    """消融实验分析器 - 复用现有基础设施"""
    
    def __init__(self, config: dict):
        self.config = config
        self.projects = self._load_all_projects()
    
    def _load_all_projects(self):
        """使用 ProjectResult 加载所有项目"""
        projects_data = {}
        for proj in self.config['projects']:
            result = ProjectResult(proj['path'])
            result.load_data()
            result.process_data()
            projects_data[proj['name']] = result
        return projects_data
    
    def calculate_natural_frequency_drift(self, project_name: str) -> dict:
        """计算固有频率漂移"""
        fn = self.projects[project_name].processed_data['fn_comped']
        metric = {
            'min': float(np.min(fn)),
            'max': float(np.max(fn)),
            'median': float(np.median(fn))
        }
        metric['drift'] = calculate_drift(metric)
        return metric
    
    def calculate_sensitivity_drift(self, project_name: str, freq_hz: float = 100) -> dict:
        """计算灵敏度漂移"""
        result = self.projects[project_name]
        result.process_data(freq_sen=freq_hz)
        sens = result.processed_data['sensitive_comped']
        metric = {
            'min': float(np.min(sens)),
            'max': float(np.max(sens)),
            'median': float(np.median(sens))
        }
        metric['drift'] = calculate_drift(metric)
        return metric
    
    def calculate_linearity(self, project_name: str) -> dict:
        """计算线性度 (1 - R²)"""
        path = f'projects/{self.projects[project_name].project_path}/linearity_by_frequency.json'
        with open(path) as f:
            data = json.load(f)
        r2_values = [item['r_squared_comped'] for item in data['linearity_by_frequency']]
        nonlinearity = [1 - r2 for r2 in r2_values]
        return {
            'mean': float(np.mean(nonlinearity)) * 100,
            'max': float(np.max(nonlinearity)) * 100,
            'min': float(np.min(nonlinearity)) * 100
        }
    
    def generate_comparison_table(self) -> str:
        """生成 Markdown 对比表格"""
        ...
    
    def generate_boxplot(self) -> str:
        """生成箱线图"""
        ...
```
