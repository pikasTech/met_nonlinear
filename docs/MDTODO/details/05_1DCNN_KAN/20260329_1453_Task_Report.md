# R3.2.1 任务报告：移除 CNNKAN 中 use_fast_model 相关代码

## 任务目标

完全移除 CNNKAN 中无用的 use_fast_model 相关代码，避免混淆。

## 执行内容

### 1. 修改 `src/models/frikan_models.py`

#### 1.1 移除 `__init__` 参数
- 移除 `use_fast_model=True` 参数
- 移除 docstring 中关于 `use_fast_model` 的说明
- 移除 `self.use_fast_model = use_fast_model` 赋值
- 移除 `self.fast_cnn` 和 `self.fast_iir` 层定义

#### 1.2 简化 `build_model` 方法
- 移除 `fast_input` Input 层
- 移除 `fast_cnn_out`、`fast_cnn_drop_out`、`fast_kan_inner_output`、`fast_output` 等变量
- 移除 `if self.use_fast_model:` 条件块（不再创建 fast_model）

#### 1.3 简化方法 docstring
- `evaluate`、`fit`、`save_weights`、`predict` 方法的 docstring 简化为单行描述

### 2. 修改 `src/core/model_engine.py`

- 移除 CNNKAN 实例化时的 `use_fast_model=self.config.USE_FAST_MODEL` 参数

## 代码变更摘要

| 文件 | 变更类型 | 变更内容 |
|------|----------|----------|
| frikan_models.py | 删除参数 | `use_fast_model=True` |
| frikan_models.py | 删除属性 | `self.use_fast_model`、`self.fast_cnn`、`self.fast_iir` |
| frikan_models.py | 简化方法 | `build_model`、`evaluate`、`fit`、`save_weights`、`predict` |
| model_engine.py | 删除参数 | `use_fast_model=self.config.USE_FAST_MODEL` |

## 验证

代码结构已简化，CNNKAN 不再包含任何 fast_model 相关代码。所有训练/评估/预测直接使用 `self.model`。
