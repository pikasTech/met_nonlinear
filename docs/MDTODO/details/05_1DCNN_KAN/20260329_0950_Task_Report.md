## 2. 实现内容

### 2.1 新增模型类 `CNNKAN`

位置: `src/models/frikan_models.py`

**架构**:
```
Input(1) → Conv1D(filters=8, kernel_size=3) → DenseKAN×6 → DenseKAN → Output(1)
```

**与 LSTM 的相似之处**:
- 不需要 system/iir_params 参数
- 直接在 __init__ 中构建模型
- 继承 BaseModel
- 简洁的构造函数

### 2.2 关键代码设计

```python
class CNNKAN(BaseModel):
    def __init__(self,
                 grid_size=5,
                 grid_range=(-1.0, 1.0),
                 spline_order=3,
                 basis_activation='silu',
                 fs=2000,
                 checkpoint_dir='data',
                 fix_scale_factor=True,
                 disable_basis_activation=True,
                 inner_kan_units=4,
                 inner_kan_layers=1,
                 use_fast_model=True,
                 cnn_filters=8,
                 cnn_kernel_size=3,
                 dropout_rate=0.2,
                 kan_log_grid=False,
                 kan_grid_expand=True,
                 save_each_epoch=False,
                 model_subcfg={}
                 ):
```

**对比 LSTM**:
```python
class LSTM(BaseModel):
    def __init__(self,
                 lstm_units=64,
                 lstm_dropout=CONF_DROPOUT,
                 activation='tanh',
                 fs=2000,
                 checkpoint_dir='data',
                 model_subcfg={},
                 ):
```

两者都不需要 system 参数，这是与 FRIKAN 的关键区别。

## 4. 模型架构详解

### 4.1 Conv1D 层
```python
self.cnn = tf.keras.layers.Conv1D(
    filters=cnn_filters,    # 8 通道
    kernel_size=cnn_kernel_size,  # 3
    strides=1,
    padding='same',
    activation='linear',
    name='cnn_filter'
)
```

### 4.2 KAN 层
- 内层: 6 个 DenseKAN 层，每层 6 个单元
- 输出层: 1 个 DenseKAN 层，输出 1 个值

### 4.3 快速模型
保留了 use_fast_model 机制，用于加速推理。

---

## 5. 与 FRIKAN 的关键差异

| 特性 | FRIKAN | CNNKAN |
|------|--------|--------|
| 滤波器层 | IIR (SIMOIIR/DIAGIIR) | Conv1D |
| 参数来源 | 从 system 辨识获取 | 可学习卷积核 |
| system 参数 | 需要 | 不需要 |
| 序列依赖 | 递归依赖 | 有限窗口 (kernel_size=3) |
