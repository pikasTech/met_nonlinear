# R1 任务报告：1DCNN_KAN 实现方案调研

**任务**: 开发一个新的模型，基于 1DCNN + KAN，来进行 MET 非线性补偿任务  
**参考模型**: FRIKANh8u6l6  
**日期**: 2026-03-29

---

## 1. 总体方案

### 1.1 目标
将 FRIKAN 中的 RNN 部分替换为 1DCNN，形成 `FRIKAN1DCNN` (或 `1DCNN_KAN`) 模型。

### 1.2 命名规则
参考 `FRIKANh8u6l6`：
- `h8`: Head units = 8，1DCNN 输入通道数（IIR 滤波器数量）
- `u6`: Inner KAN units = 6
- `l6`: Inner KAN layers = 6

## 2. FRIKAN 架构分析

### 2.1 FRIKAN 模型结构
```
Input(1) → IIR层(SIMOIIR/DIAGIIR) → Inner KAN层(DenseKAN×6) → Output KAN(DenseKAN) → Output(1)
              ↓
         h8=8通道输出
```

### 2.2 IIR 层实现
位于 `src/experimental/mimoiir.py`：
- 使用 `SimpleRNN(units=4)` 递归实现 IIR 差分方程
- 状态空间矩阵 A、B、C 实现滤波器特性

### 2.3 KAN 层实现
位于 `src/tfkan/layers/dense.py`：
- `DenseKAN`: 全连接 KAN 层
- `Conv1DKAN`: 已存在，可直接使用

## 3. 1DCNN 替换方案

### 3.1 核心思路
将 IIR 层的递归结构替换为 1DCNN 的非递归卷积结构：
- IIR: 递归滤波，依赖历史输出 y[n-1], y[n-2]
- 1DCNN: 非递归卷积，仅依赖输入 x[n], x[n-1], x[n-2]

### 3.2 替换策略

**方案 A（推荐）**: 用 1DCNN 直接替换 IIR 层
```python
# 1DCNN: 1输入，8输出
self.cnn = tf.keras.layers.Conv1D(
    filters=8,  # h8 = 8通道输出
    kernel_size=3,
    strides=1,
    padding='same',
    activation='linear'
)
```

**方案 B**: 完整重写，直接继承 BaseModel 构建 1DCNN + KAN 模型
```python
class FRIKAN1DCNN(BaseModel):
    def __init__(self, *args, cnn_kernel_size=3, **kwargs):
        # 直接用 1DCNN + KAN 构建模型
```

### 3.3 模型维度变化
```
Input(1) → Conv1D(h8) → Inner_KAN(Conv1DKAN×6) → Output_KAN(DenseKAN) → Output(1)
```

### 4.2 关键代码修改

#### 4.2.1 在 frikan_models.py 添加 FRIKAN1DCNN 类
```python
class FRIKAN1DCNN(FRIKAN):
    def build_filter_layer(self):
        return tf.keras.layers.Conv1D(
            filters=8,  # h8 = 8通道输出
            kernel_size=3,
            strides=1,
            padding='same',
            activation='linear'
        )
```

#### 4.2.2 config.json 配置
```json
{
    "use_model": "FRIKAN1DCNN",
    "H_UNITS": 8,
    "INNER_KAN_UNITS": 6,
    "INNER_KAN_LAYERS": 6,
    "GRID_SIZE": 8,
    "SPLINE_ORDER": 2,
    "basis_activation": "tanh"
}
```
