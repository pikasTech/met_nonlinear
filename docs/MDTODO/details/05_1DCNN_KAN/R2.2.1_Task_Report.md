### 类结构
- **类名**: `CNNKAN`
- **基类**: `BaseModel`
- **架构**: `Input(1) → Conv1D(filters=8, kernel_size=3) → DenseKAN×N → DenseKAN → Output(1)`

### 关键组件
1. **Conv1D 层**: `cnn_filters=8`, `cnn_kernel_size=3`, `padding='same'`
2. **快速模型**: `fast_cnn` 与 `cnn` 结构相同，支持 `use_fast_model` 机制
3. **KAN 层**: 输出层 `DenseKAN(units=1)` + 内层 `DenseKAN×inner_kan_layers`
4. **Dropout**: 支持可配置的 dropout

### 构造函数参数
