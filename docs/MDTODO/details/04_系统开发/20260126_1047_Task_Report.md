## 5. 预期问题与解决方案

### 5.1 常见问题

| 问题 | 原因 | 解决方案 |
|------|------|----------|
| ModuleNotFoundError | 模块未正确迁移或 sys.path 未设置 | 检查 cli.py 的 sys.path 添加代码 |
| ImportError | 循环导入或模块依赖问题 | 检查模块间导入关系 |
| subprocess 导入失败 | 子进程未继承 PYTHONPATH | 设置环境变量 |
| IDE 路径错误 | IDE 未刷新缓存 | 重启 IDE 或刷新项目 |

### 5.2 调试技巧

