## 5. 数学原理

### 5.1 理想SVF传递函数

**位置**: `visualization/wnet5_circuit_validator.py:1403-1426`

分母多项式：
$$D(s) = s^2 + \frac{\omega_0}{Q}s + \omega_0^2$$

三种滤波输出：
- 高通: $H_{HP}(s) = \frac{s^2}{D(s)}$
- 带通: $H_{BP}(s) = \frac{\frac{\omega_0}{Q}s}{D(s)}$
- 低通: $H_{LP}(s) = \frac{\omega_0^2}{D(s)}$

### 5.2 带误差SVF计算

**位置**: `visualization/wnet5_circuit_validator.py:771-818`

$$H_{measured}(j\omega) = G_{measured}(\omega) \times e^{j\theta_{theory}(j\omega)}$$

其中 $G_{measured}(\omega)$ 从Excel实测数据读取。

### 5.3 时域递推仿真

**位置**: `spice_simulator/circuit_svf.py:384-458`

```python
hp = input_signal[j] - lp_z1 - F2 * bp_z1  # 高通
bp = bp_z1 + F1 * hp * dt                   # 带通
lp = lp_z1 + F1 * bp_z1 * dt                # 低通
```

## 8. 架构分析

### 8.1 WNET5模型结构

```
WNET5q1h2u6l3
├── SVFLayer (索引0)
│   ├── center_freqs: 中心频率列表
│   └── quality_factors: Q因子列表
├── Dense_Layer_Model_1 (索引1)
├── Dense_Layer_Model_2 (索引2)
├── Dense_Layer_Model_3 (索引3)
└── Output_Layer_Model (索引4)
```

### 8.2 SVF输出通道

每个SVF滤波器产生3个输出通道：
- `out1`: 高通 (HP)
- `out2`: 带通 (BP)
- `out3`: 低通 (LP)

多个SVF滤波器输出依次排列：HP0, BP0, LP0, HP1, BP1, LP1, ...
