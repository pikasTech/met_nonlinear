def _init_filters(self):

#### 4.2.3 硬件感知损失函数

**新增文件**：`src/core/hardware_aware_loss.py`

**设计**：
```python
def hardware_aware_mse_loss(y_true, y_pred, svf_layer=None, config=None):
    """
    硬件感知MSE损失函数

    包含两部分：
    1. 标准MSE损失
    2. 硬件误差正则化项（可选）
    """
    nominal_loss = tf.reduce_mean(tf.square(y_true - y_pred))

    if config and config.get('error_weight', 0) > 0:
        # 计算理想输出与实测输出的差异
        ideal_output = svf_layer.forward_ideal(x)
        measured_output = svf_layer.forward_measured(x)
        hardware_loss = tf.reduce_mean(tf.square(ideal_output - measured_output))
        return nominal_loss + config['error_weight'] * hardware_loss

    return nominal_loss
```
