## 2. 实现细节

### 2.1 硬件感知配置管理类

**文件**: `src/core/hardware_aware_config.py`

#### 2.1.1 类设计

```python
class HardwareAwareConfig:
    """硬件感知配置管理类

    功能：
    - 从 model_subcfg 或 JSON 文件加载配置
    - 配置验证和参数检查
    - 参数传递到模型层
    """

    def __init__(self, config: Optional[Dict] = None):
        self.config = config or {}
        self.hat_config = self.config.get('hardware_aware_training', {})
```

#### 2.1.2 核心方法

| 方法 | 功能 |
|------|------|
| `is_enabled()` | 检查硬件感知训练是否启用 |
| `use_fitted_params()` | 检查是否使用拟合参数 |
| `get_error_weight()` | 获取硬件误差权重 |
| `get_svf_params(layer_name)` | 获取指定层的SVF参数 |
| `validate_params()` | 验证参数有效性 |

### 2.2 拟合参数生成脚本

**文件**: `scripts/generate_fitted_params.py`

#### 2.2.1 功能说明

- 从 Excel 文件读取实测 SVF 频率响应数据
- 使用 `scipy.optimize.curve_fit` 进行曲线拟合
- 提取 f0, Q, gain 参数
- 保存为 JSON 格式

#### 2.2.2 使用方法

```bash
python scripts/generate_fitted_params.py \
    --input exam_data/20251230_SVFNET_SVF_ONLY/20251230_SVF_ONLY.xlsx \
    --output data/svf_fitted_params.json
```

#### 2.2.3 拟合函数

```python
def svf_magnitude_response(f: np.ndarray, f0: float, Q: float,
                           gain: float = 1.0) -> np.ndarray:
    """SVF 频率响应幅度函数（二阶低通滤波器模型）

    H(jω) = gain / sqrt((1 - (ω/ω0)^2)^2 + (ω/(ω0*Q))^2)
    """
```
