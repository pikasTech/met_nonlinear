**严重程度**: 中

**问题描述**:
在配置格式设计部分（4.1.1），`training_mode` 中使用了 `dense_trainable`，但在数据流总体架构图（3.1）中使用了 `trainable_dense_only`，两者不一致。

**R4.3.1 报告原文**:
```json
// 4.1.1 配置格式

  "dense_trainable": true  // <-- 使用 dense_trainable
}

// 3.1 数据流架构图
"trainable_dense_only": true  // <-- 使用 trainable_dense_only
```

**建议修复**:
统一使用 `trainable_svf` 和 `trainable_dense`，明确表达"哪些层可训练"的语义。

**修改后的配置格式**:
```json

  "trainable_svf": false,
  "trainable_dense": true
}
```

### 4.2 滤波器类型确定逻辑缺失

**严重程度**: 中

**问题描述**:
在 `_init_filters_hardware_aware` 方法和 `compute_iir_coefficients` 函数中，使用了 `determine_filter_type(i)` 函数，但该函数在报告中未定义具体实现逻辑。

**R4.3.1 报告原文**:

```python
    # ...
    filter_type = determine_filter_type(i)  # <-- 函数未定义

def compute_iir_coefficients(f0: float, Q: float, filter_type: str, fs: int) -> dict:
    # ...
    if filter_type == 'highpass':
        # ...
```

**建议补充**:
WaveNet5 的 SVF 层每个中心频率对应 3 个滤波器（highpass, bandpass, lowpass），滤波器类型应按以下规则确定：
- 索引 % 3 == 0: highpass
- 索引 % 3 == 1: bandpass
- 索引 % 3 == 2: lowpass

**补充代码建议**:
```python
def determine_filter_type(channel_index: int) -> str:
    """
    根据通道索引确定滤波器类型

    WaveNet5 SVF 层每个中心频率对应 3 个滤波器:
    - 索引 0, 3, 6, ...: highpass (高通)
    - 索引 1, 4, 7, ...: bandpass (带通)
    - 索引 2, 5, 8, ...: lowpass (低通)

    Args:
        channel_index: 通道索引

    Returns:
        str: 滤波器类型 ('highpass', 'bandpass', 'lowpass')
    """
    filter_types = ['highpass', 'bandpass', 'lowpass']
    return filter_types[channel_index % 3]
```

```python
        fitted = channel.get('fitted_params', {})

```

**建议补充**:
IIR 滤波器的稳定性取决于极点位置，需要验证 IIR 系数是否会导致不稳定滤波器。

**补充代码建议**:
```python
def validate_iir_stability(a1: float, a2: float, b0: float, b1: float, b2: float) -> tuple[bool, str]:
    """
    验证IIR滤波器系数的稳定性

    对于二阶IIR滤波器，稳定条件是:
    - |a2| < 1 (无极点在单位圆外)
    - |a1| < |a2| + 1 (无极点在单位圆上)

    Args:
        a1, a2: 分母系数
        b0, b1, b2: 分子系数

    Returns:
        tuple: (是否稳定, 状态消息)
    """
    import numpy as np

    # 检查系数范围
    if abs(a2) >= 1.0:
        return False, f"系数 a2={a2:.6f} 超出稳定范围 (|a2| < 1)"

    if abs(a1) >= abs(a2) + 1:
        return False, f"系数组合不稳定: a1={a1:.6f}, a2={a2:.6f}"

    return True, "滤波器稳定"


def validate_measured_params_iir(measured_params: dict) -> tuple[bool, list]:
    """
    验证所有通道的IIR系数稳定性
    """
    warnings = []
    svf_channels = measured_params.get('svf_channels', [])

    for i, channel in enumerate(svf_channels):
        coeffs = channel.get('iir_coefficients', {})
        a1 = coeffs.get('a1', 0)
        a2 = coeffs.get('a2', 0)
        b0 = coeffs.get('b0', 0)
        b1 = coeffs.get('b1', 0)
        b2 = coeffs.get('b2', 0)

        is_stable, message = validate_iir_stability(a1, a2, b0, b1, b2)
        if not is_stable:
            warnings.append(f"通道 {i}: {message}")

    return len(warnings) == 0, warnings
```
