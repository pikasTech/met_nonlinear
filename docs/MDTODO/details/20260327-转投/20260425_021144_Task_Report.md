# R21 任务报告：清理论文泛化定性评估内容

## 任务理解

R21 的核心要求是清理正文中缺少外推实验支撑的泛化、外部验证和未来扩展类定性表述。本文当前证据链主要来自单一 MET 样品、25°C 环境、自采正弦激励和频率--震级响应矩阵；因此正文应把结论限定在该实验矩阵内，避免把补偿效果扩展到跨温度、跨传感器、真实地震波形、外部样品或野外场景。

## 全文论证主线梳理

- Introduction 的主线是：MET 幅频耦合非线性限制动态范围，力反馈增加硬件复杂度和稳定裕度风险，因此引出前馈补偿；随后从数据集覆盖范围、物理先验融合、KAN 可解释非线性映射和低算力部署建立 Wiener-KAN 的必要性。
- Background and Principles 的主线是：从 MET 结构、电化学-机械耦合和幅值相关局部频响漂移出发，给出后续 Wiener 线性层初始化和幅频耦合补偿的物理依据。
- Experimental Design and Dataset 的主线是：说明单一实验装置、频率--震级扫描矩阵、理想参考系统、训练/验证/测试划分和固定训练策略，是当前指标的证据边界。
- Proposed Method 的主线是：前馈框架由 Wiener 线性动态补偿和约束 KAN 静态非线性映射串联，局部频响先验、频率注入和 AFMAE 分别支撑频域补偿能力。
- Experiments and Results 的主线是：在统一 MET 补偿协议下比较 Wiener-KAN 与传统 Wiener、时序网络和 physics-inspired baseline，并补充嵌入式部署评估。
- Discussion/Conclusion 应承接上述证据范围，强调当前实验矩阵内的补偿效果、数据开放和部署可行性，而不展开缺少实验证据的外部泛化评价。
