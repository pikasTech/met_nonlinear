## 三、转投修订策略

### 3.1 论文定位调整

根据 `FRIKAN_PRINCIPLE.md` 的分析，转投时应采用以下策略：

#### 创新点收缩
- **不再声称**：补偿精度显著提升（vs LSTM/GRU）
- **核心声称**：计算效率优势 + 特定场景下的有效性

#### 模型重构
```
原架构：FRIKAN = PIKAN + FRIRNN
新架构：Wiener-KAN = 线性RNN部分 + KAN非线性部分
```

**重构依据**：Wiener模型在非线性系统识别领域有成熟理论基础，将FRIKAN重新诠释为Wiener-KAN架构，可建立与经典理论的联系，增强学术可信度。

### 3.2 必做实验补充

#### 实验1：物理约束消融实验

| 实验组 | 奇对称性 | 正定性 | 预期结论 |

1. **实验验证不够系统**：缺乏完整的消融实验和对比实验
