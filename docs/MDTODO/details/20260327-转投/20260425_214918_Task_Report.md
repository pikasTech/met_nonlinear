### 1. LUT 与线性插值方法前移到 Method

在 `Proposed Method` 中新增 `LUT-Based KAN Inference and Embedded Cost Model` 小节，内容包括：

- KAN 激活函数离线采样为 LUT 的定义；
- 最近邻查表公式；
- 一阶线性插值公式；
- KAN LUT 路径与 MLP 矩阵乘加路径的计算结构差异；
- ADD--MUL--MAP 静态 Compute Cost 公式；
- STM32F405 对应的 add 归一化权重由 `valCostModel...` 宏注入。

该调整把原本偏结果描述的 LUT 原理图和 LUT 解释迁移到方法章节，使 Results 部分只保留部署实测与取舍结论。

### 2. Results 改为三大块叙事

将 `Experiments and Results` 重组为三类结果：

1. `Wiener-KAN Compensation and On-Board Evaluation`
   - 聚焦提出的 Wiener-KAN 主模型；
   - 展示频率响应漂移抑制、输入输出线性度、时域波形；
   - 单独加入 `On-board inference test flow`，说明 C 导出、QEMU、KEIL 真机、串口基准循环、Flash/RAM 采集流程；
   - 保留同一 Wiener-KAN 权重下 LUT nearest 与 LUT + linear interpolation 的板端速度/误差取舍表。

2. `Comprehensive Comparison with Baseline Models`
   - 汇总 Wiener、1DCNN、TCN、RNN、GRU、LSTM、LSTMTransformer 与 Wiener-KAN 的六维指标横评；
   - 将基线模型讨论集中到横评小节，避免在主模型效果展示段落中过度发散。

3. `Ablation Studies of Wiener-KAN Components`
   - 新增总览表 `tab:ablation_overview`；
   - 按组展开 KAN 映射、物理约束、Wiener 前端、超参数、Loss 函数消融；
   - 按 R23.1 要求把消融先组织为总表，再逐个小节分析。

### 3. 补齐消融实验正文结构

新增或重排了以下消融内容：

- `Activation-function and mapping-structure ablation`
  - 比较 B-spline KAN、ReLU、tanh、Sigmoid 和 FRIMLP；
  - 明确 KAN vs MLP 的结构消融位置。

- `Physical sign and symmetry constraints`
  - 保留无约束、Odd-only、Positive-only、Odd + positive 矩阵；
  - 将其纳入统一消融章节。

- `Wiener front-end initialization and trainability`
  - 新增系统先验冻结、随机冻结、系统先验可训练、随机可训练四种 IIR 前端消融表；
  - 使用 `values.tex` 宏维护关键指标。

- `Hyperparameter sensitivity`
  - 保留 h/u/l/grid/order 单因素敏感性分析；
  - 明确说明 spline grid 与 spline order 在 LUT 导向的在线操作计数中保持同一成本范围，是 KAN 适配边缘部署的重要特性。

- `Loss-function ablation`
  - 移入消融大块末尾，和其它内部组件消融形成统一结构。
