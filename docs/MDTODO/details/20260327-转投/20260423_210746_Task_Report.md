## 1. 理论提炼结果

本轮主要抽取并整理了以下两份上游文稿中的非线性机理：

- `C:\work\big-paper3\3.正文\第2章-电化学检波器传感原理和频率响应分析\2.4-频率响应影响因素与关键问题.md`
- `C:\work\big-paper3\3.正文\第5章-电化学检波器震级相关的频响漂移补偿技术\5.2-震级对电化学检波器频响的影响分析.md`

最终收敛出的核心结论是：

1. 大信号非线性不是单一来源，而是机械几何非线性、流体惯性非线性、离子输运乘法非线性和 Butler-Volmer 指数反应动力学共同作用的结果；
2. 这些机制共同导致了“幅值依赖的频率响应漂移”和“频率依赖的非线性失真”两个同时存在的问题；
3. 因而后续补偿模型需要同时覆盖局部动态迁移与静态非线性调制，不能只写成单一静态映射。

这部分结论已经被吸收进专题过程稿和总稿 `2.2 Wiener-KAN 结构` 的前导段落中。

## 3. 复现结果摘要

### 3.1 中心频率拟合

- 仿真范围：`33.4406 - 88.1083 Hz`
- 目标范围：`34.2500 - 93.4092 Hz`
- `MAE = 1.8713 Hz`
- `RMSE = 2.3356 Hz`
- 最大绝对误差：`5.7084 Hz`

### 3.2 100 Hz 灵敏度拟合

- 仿真范围：`55.8681 - 197.4977`
- 目标范围：`60.6216 - 203.3722`
- `MAE = 4.5326`
- `RMSE = 5.5307`
- 最大绝对误差：`10.5859`

### 3.3 当前结论

这些结果足以支撑一个克制但清晰的结论：三支路并联 Wiener 等效模型已经能够在整体趋势上重建电化学检波器的幅频耦合非线性，尤其能够同时捕捉：

- 随输入幅值升高而持续右移的峰频；
- 随输入幅值升高而抬升的 100 Hz 通带灵敏度；
- 不同幅值区间由不同局部工作点主导的动态迁移过程。

因此，这一模型适合作为 Wiener-KAN 前端先验的上游解释层，而不是直接被写成最终部署模型本身。
