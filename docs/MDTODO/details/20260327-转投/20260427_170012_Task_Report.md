# R24.12 任务报告：Ablation Studies 结构分组说明与命名统一

## 任务目标

R24.12 要求补齐 `Ablation Studies` 章节中对原 `Mapping Group` 的解释，并将该分组改写为更贴切的 `Struct Group`。同时需要：

- 明确两类结构消融分别替换了 Wiener-KAN 的哪一部分；
- 将 `FRIMLP` 在论文口径中统一改写为 `Wiener-MLP`；
- 更新相关表格、结果快照和图表原始数据中的命名；
- 保持当前论文 pipeline 的数据、表格、正文和 PDF 输出一致。

## 实施内容

### 1. 正文补充

在 `docs/paper/latex/main.tex` 的 `Ablation Studies of Wiener-KAN Components` 小节中完成以下调整：

- 将引导语中的“非线性映射”改为“结构变体”；
- 新增 `Structural ablation of the Wiener and KAN blocks` 小节；
- 明确说明：
  - `CNN-KAN` 用卷积前端替代 Wiener 线性块，并保留 KAN 非线性映射；
  - `Wiener-MLP` 保留 Wiener 前端，并用 MLP 替代带物理约束的 KAN 映射；
- 增加对两类结构消融结果的定性解释，使读者能够直接理解 `Struct` 分组在表中的物理含义。

### 2. Paper pipeline 命名统一

在 `docs/paper/config.json` 与 `docs/paper/src/paper_pipeline.py` 中完成以下统一：

- `CNNKAN` -> `CNN-KAN`
- `FRIMLP` -> `Wiener-MLP`
- `Mapping` -> `Struct`
- `Conv front + KAN` -> `Conv front replaces Wiener front`
- `MLP mapping` -> `MLP replaces KAN mapping`

同时同步修正了 `values.tex` 生成时依赖的结构消融标签映射，避免后续重生成时因旧标签失配而报错。

## 额外修复

在执行 LaTeX 验证时，发现 `docs/paper/latex/main.tex` 引言部分存在一处已存在的公式断裂：

- `\valXuPeakSensitivity~$\text{V} \cdot \text{s}` 后半句缺失，导致 XeLaTeX 在第 123 行触发 `Missing $ inserted`

这是一处编译阻断修复，不改变 R24.12 的结构消融结论。
