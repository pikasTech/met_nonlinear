# 总目标

为 src/ 增加单元测试，测试覆盖率提高到 80%。

要求：

1. 全量测试时间控制在 3 分钟以内，支持并行测试。
2. 如果 1 分钟内测试没有 console log 输出，则视为卡死失败。
3. 测试集成到 cli.py test 命令中。

## 开发必读

测试环境：tf26 

环境启动方法：

```bash
conda.bat run --no-capture-output -n tf26 python cli.py xxx
```

**关键参数**: `--no-capture-output` - 避免编码问题
