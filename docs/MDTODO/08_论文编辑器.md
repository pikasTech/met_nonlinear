## R1 [completed]

用 ts 前后端 + react 开发一个 latex 论文快速预览编辑器，要求：

1. 快速预览，快速编译为 markdown，并在 webui 上面展示预览效果，不要求严格支持 latex 的格式控制，只做简单的文本、标题等解析。
2. 支持渲染 latex 数学公式，支持简单的宏替换。
3. 支持导入其他 .tex，例如 main.tex 导入 values.tex，实现里面的数值宏解析。
4. 能显示图片。
5. 用 `docs\paper\latex\main.tex` 做测试。
6. 要能够测量编辑器的实时运行的性能，有性能分析面板，便于后续性能优化。
7. 有源代码模式，能直接编辑 latex 代码，能实时预览效果，实现左右分屏编辑，左屏编辑，右屏预览，同步滚动。
8. 左侧有大纲导航，能分级导航文档结构，点击导航能跳转到对应位置。
9. 要用 playwright 做端到端测试。
