# MN 最后冻结投稿

## 冻结原则

本文件用于最后一轮 MN 投稿前收口。默认不再继续改动论文实质内容：不新增实验、不替换 canonical 结果、不改变指标口径、不重写核心贡献。后续只允许三类机械性工作：

- 投稿格式合规：正文词数、图表数量、补充材料分流、纯英文 clean 源。
- 投稿材料补齐：Data/Code Availability、Supplement、cover letter、图片和参考文献文件。
- 最终验证：编译、版面、缺图缺宏、旧术语和内部痕迹扫描。

当前冻结基线：

- 主稿源：`docs/paper/latex/main.translated.tex`
- 编译 PDF：`docs/paper/latex/build/main.translated.pdf`
- 数据快照：`docs/paper/data/results.json`
- 数值宏：`docs/paper/latex/values.tex`
- 图源：`ex_projects/plot/**/data/*.png`

## 当前 MN 格式缺口

根据 MN Guide for Authors 的 Article 常规口径，主文应接近：

- 摘要：不超过约 250 词。
- 正文：约 6000 词。
- 图表：最多约 8 个 display items。
- 参考文献：最多约 50 篇。

当前英文稿的快速盘点结果：

- 摘要约 236 词，基本合规。
- 正文粗略约 8300 词，需压缩约 2300 词。
- 主文约 13 图 + 5 表 = 18 个 display items，需压缩到约 8 个。
- 参考文献约 29 篇，基本合规。

因此最后冻结阶段的核心不是重做科学内容，而是把已经形成的证据链重新分配到主文和 Supplement。

## 正文超出解决策略

目标：在不改变结论、不改变实验结果、不改变指标口径的前提下，把主文正文压到约 6000 词。

建议采用“转移优先、压缩其次、改写最少”的策略：

- 优先转移到 Supplement：
  - 详细物理机理推导。
  - 并联 Wiener 等效模型的公式细节和拟合过程。
  - AFMAE 的实现细节与长公式解释。
  - LUT 最近邻 / 线性插值的底层部署流程细节。
  - 超参数敏感性逐轴解释。
  - 消融实验中每个变体的长段落解释。

- 主文保留：
  - 问题背景与 gap。
  - 数据集构建的核心事实。
  - Wiener-KAN 架构的核心思想。
  - 主补偿结果。
  - 横评摘要。
  - 消融总表和最关键结论。
  - 部署可行性核心结果。
  - limitations 和 Data Availability。

- 需要直接压缩的段落：
  - Introduction 中重复出现的“力反馈限制、边缘实时、物理先验缺失”叙述。
  - Background and Principles 中多物理公式后的解释性长段。
  - Proposed Method 中 KAN、AFMAE、LUT 三段的实现细节。
  - Results 中消融分组的逐项解释。

- 不建议压缩或移动：
  - Abstract。
  - Main benchmark 的关键数值。
  - Data Availability。
  - Limitations / validation boundary。

最小可接受路线：不追求漂亮压到 6000 以内，而是把正文降到约 6500--7000 词，并把显示项目压到 8 个以内；这样即使编辑要求再压，也不会再牵动实质内容。

## 图表超出解决策略

目标：主文最多保留约 8 个 display items，其余转入 Supplement。当前 18 个 display items 必须分流。

建议主文保留 8 项：

- MET structure + readout / experimental setup + dataset workflow 合并图：证明传感器对象和数据来源。
- Nonlinear frequency response / amplitude-dependent drift 图：证明问题存在。
- Wiener-KAN framework 图：证明方法结构。
- Main compensation result 图：证明主方法有效。
- Main benchmark table：承载横评核心数值。
- Horizontal summary figure：承载物理指标、吞吐率、雷达图和收敛摘要。
- Ablation overview table：承载组件贡献证据。
- Embedded inference figure/table 合并项：证明板端部署可行。

建议转 Supplement：

- 详细物理机理示意图。
- 并联 Wiener 等效建模图。
- KAN 单神经元 toy example。
- Local transfer slices 图。
- LUT principle 详细流程图。
- Hyperparameter sensitivity 全图或其逐轴细节。
- Loss convergence 细节曲线。
- 任何重复承载同一结论的表格。

如果用户坚持完全不动主文图表，风险是：投稿时可能被格式检查或编辑要求返修。最快合规方案仍然是“主文保留 8 项，Supplement 收纳其余图表”，这是格式层面的移动，不改变论文实质内容。
