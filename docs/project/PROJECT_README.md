# WNET5最优变种+PS-5/360检波器数据项目

## 项目概述

本项目基于WNET5的最优变种**EFF2_A1**配置，整合了WaveNet5模型的最优参数和PS-5/360检波器数据配置。EFF2_A1是经过大量实验验证的最优配置，具有最佳的建模性能和稳定性。

## 最优配置来源 - EFF2_A1

**EFF2_A1**是当前最优的WNET5变种，经过系统性实验验证，具有以下核心优势：

1. **多频段覆盖**: 6个优化中心频率点，覆盖8-180Hz全频段
2. **分级品质因数**: 1.5-5.0递进式Q值，优化不同频段特性
3. **增强后处理**: 14个隐藏单元的密集层，提供更强的建模能力
4. **密集偏置**: 启用密集层偏置，提升模型灵活性

## 核心配置参数

### 模型架构 - WaveNet5 (基于EFF2_A1最优配置)

**最优参数组合**：
- `init_center_freqs`: [8, 25, 50, 85, 120, 180] - 6个优化频率点，覆盖全频段
- `init_quality_factors`: [1.5, 2.0, 2.5, 3.0, 4.0, 5.0] - 分级品质因数，优化各频段特性
- `post_dense`: true - 后处理密集层增强
- `post_dense_activation`: "relu" - ReLU激活函数
- `post_dense_units`: 14 - 14个隐藏单元（增强建模能力）
- `post_dense_layers`: 3 - 3层后处理
- `use_dense_bias`: true - 启用密集层偏置
- `kernal_units`: 6 - 核单元数

### 核心创新点

1. **多频段中心频率**: [8, 25, 50, 85, 120, 180] Hz
   - 比传统2频段配置覆盖更宽频域
   - 频率间隔优化，避免频段重叠

2. **分级品质因数**: [1.5, 2.0, 2.5, 3.0, 4.0, 5.0]
   - 低频低Q值，高频高Q值
   - 针对不同频段特性优化谐振响应

3. **增强密集层**: 14个隐藏单元
   - 比标准6单元配置提供更强建模能力
   - 适用于复杂非线性特性建模

4. **密集偏置**: 启用密集层偏置
   - 提升模型表达灵活性
   - 增强特征学习能力
