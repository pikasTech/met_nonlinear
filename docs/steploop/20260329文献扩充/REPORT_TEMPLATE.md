## 报告模板规范

本文件用于统一文献调研各阶段报告的结构，保证后续文档可追溯、可比较、可复查。

## 通用要求

- 所有报告必须使用中文编写
- 文件名应简洁表达本轮主题，例如：`Wiener模型文献调研.md`、`KAN网络分析.md`
- 报告必须写明日期、轮次主题、覆盖范围、产出文档
- 如使用子代理并行，必须在报告中交代拆分维度和汇总方式
- 报告中涉及的文献、方法、结论，必须尽量给出原始链接或 DOI

## 最小必需字段

即使报告需要压缩，也不能删除以下信息：

- 日期
- 阶段（调研/分析/综合）
- 覆盖范围或分析对象
- 使用的数据库/来源
- 结论或发现结果
- 对哪些文档产生影响
- 原始链接或 DOI
