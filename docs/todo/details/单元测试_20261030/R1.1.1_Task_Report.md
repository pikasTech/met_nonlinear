## 使用方法

### 基本测试运行

```bash
# 使用默认配置运行测试
conda.bat run --no-capture-output -n tf26 python cli.py --test

# 指定测试路径
conda.bat run --no-capture-output -n tf26 python cli.py --test --test-path src/core/tests

# 设置超时时间
conda.bat run --no-capture-output -n tf26 python cli.py --test --test-timeout 60

# 禁用并行测试
conda.bat run --no-capture-output -n tf26 python cli.py --test --no-parallel

# 设置并行 worker 数量
conda.bat run --no-capture-output -n tf26 python cli.py --test --test-workers 8
```

### 直接使用 pytest

```bash
# 直接使用 pytest-parallel
conda.bat run --no-capture-output -n tf26 python -m pytest src/tests -v --workers 4

# 带超时
conda.bat run --no-capture-output -n tf26 python -m pytest src/tests --timeout 300
```
