## 经验总结

### 1. 模块依赖处理
- `figure_paper` 模块依赖 `paper.fig_process` 等外部模块，使用 `MagicMock` 进行模拟
- `data_viewer` 模块依赖 `calibration_analyzer`，测试时需要正确处理模块导入

### 2. 临时文件管理
- 使用 `tempfile.mkdtemp()` 创建临时测试目录
- 使用 `shutil.rmtree()` 确保测试后清理

### 3. matplotlib 后端处理
- 测试图表生成功能时使用 `matplotlib.use('Agg')` 设置非交互式后端
- 使用 `patch('matplotlib.pyplot.show')`弹出

### 4. Mock 对象属性
- 使用 防止图形窗口 `MagicMock()` 创建模拟对象时，需要显式设置 `name` 属性以支持 `os.scandir` 等需要 name 属性的场景
