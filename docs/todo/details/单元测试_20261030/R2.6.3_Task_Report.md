## 修复的问题

1. **plt.show() 超时**: 使用 `@patch` 模拟 `matplotlib.pyplot.show` 防止测试超时
2. **权重长度错误**: 修正 `kan_lut_siso` 测试中的权重长度（5 元素 = grid_size + spline_order）
3. **DIAGIIR 多单元问题**: 限制 `run_performance_test` 测试使用 units=1 避免已知的形状兼容性问题
