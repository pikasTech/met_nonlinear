```

R2.7.7 报告覆盖率:  44%
差异来源:
  - 包含/排除规则不同
  - 计算方法不同 (行覆盖率 vs 语句覆盖率)
  - 过滤的代码行数不同
```

#### 根本原因 3：代码结构变化

- exam_class.py: 1404 行语句
- waveprocessor.py: 254 行语句
- 报告中的 72% 覆盖率意味着需要覆盖约 1011 行

1. **覆盖率报告需要验证**
   - 子任务报告的覆盖率数据与实际测量存在巨大差异
