### 5.1 Mock 技术应用

```python
# 示例：Mock RawRead
class MockRawRead:
    def __init__(self, filepath, dialect='ngspice'):
        self._traces = {}

    def get_trace(self, name):
        if name not in self._traces:
            self._traces[name] = MockTrace(name)
        return self._traces[name]
```
