# 偏置补偿实施要点

## 快速参考

### 问题描述
- 加法器已加入bias电压，但运放非理想特性导致实际输出有误差
- 需要在输出端补偿，使得直流电平回到理想bias值

### 解决方案
在输出端串联无源高通滤波器，电阻连接到bias电压而非地：

```
[前级输出]----C----+----[补偿后输出]
                   |
                   R
                   |
              [bias电压]
```

### 推荐参数

| 截止频率 | R值 | C值 | 建立时间(99%) | 适用场景 |
|---------|-----|-----|--------------|----------|
| 0.01Hz | 1MΩ | 16μF | 80s | 极低频信号 |
| 0.1Hz | 1MΩ | 1.6μF | 8s | 推荐默认值 |
| 1Hz | 160kΩ | 1μF | 0.8s | 快速响应 |

### 注意事项

1. **bias节点需要低阻抗驱动**（如运放输出或去耦电容）
2. **前级运放需要足够的输出电流能力**
3. **建立时间为5τ**，需根据系统要求选择

### 验证步骤

1. 测量无补偿时的直流偏移
2. 加入补偿电路后测量稳态直流电平
3. 检查信号带宽内的频率响应
4. 验证瞬态响应和建立时间
